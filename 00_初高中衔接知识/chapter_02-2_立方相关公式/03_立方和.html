立方和

a3+b3 是两个完整立方相加，目标是分解。关键不是一眼猜出因子，而是先找到自然的一次因子，再让二次因子的中间项承担抵消任务。

01先判断对象

两端都要是完整立方

能用立方和公式，首先要看两端是否都能写成三次方。

x3+27 可以看成 x3+33，所以是立方和。

x3+9 在当前整式因式分解范围内不能直接看成立方和。

这一步先分清对象：它不是 (a+b)3，所以不要展开成四项。

02推出公式

先找一次因子，再设中间项

一次因子不是凭空猜出来的。先问：a3+b3 什么时候会等于 0？

如果 b = -a a3 + (-a)3 = 0 b = -a，也就是 a+b = 0

所以分解时最自然先尝试的一次因子是 a+b。这还不是最终公式，只是找到一个候选因子；接下来要乘回去检查它是否真的合适。

拿出一次因子 a+b 后，剩下的因子应是二次式。要乘出首项 a3，二次因子的首项应是 a2；要乘出末项 b3，末项应是 b2。中间项还不知道，先写成 kab。

a3+b3 = (a+b)(a2 + kab + b2) (a+b)(a2 + kab + b2) = a3 + ka2b + ab2 + a2b + kab2 + b3 = a3 + (k+1)a2b + (k+1)ab2 + b3

左边只有 a3 和 b3，所以混合项必须消失：

k + 1 = 0 k = -1 a3+b3 = (a+b)(a2 - ab + b2)
二次因子的中间项是 -ab，因为它负责抵消乘回去出现的两类混合项。
03抵消模型

把乘开后的混合项拖到一起

右边乘开后会出现六个体积项。真正决定公式的是两对混合项：一个正的和一个负的同形体积块相互抵消，最后只剩 a3+b3。

04例子检查

分解 x³ + 27

先写成立方和：

x3 + 27 = x3 + 33 = (x+3)(x2 - 3x + 9)

乘回去检查时，3x2 与 -3x2 抵消，9x 与 -9x 抵消，最后只剩 x3+27。
