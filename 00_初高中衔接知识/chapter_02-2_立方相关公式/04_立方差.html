立方差

a3-b3 是两个完整立方相减，目标是分解。先从“什么时候等于 0”找到一次因子，再推出二次因子为什么全是加号。

01先判断对象

看清真正被立方的是谁

立方差要求两端都是完整三次方。比如：

8x3 - 125 = (2x)3 - 53

这里的 a 是整个 2x，不是单独的 x。

02推出公式

先找一次因子，再设中间项

一次因子也不是硬背出来的。先问：a3-b3 什么时候会等于 0？

如果 a = b a3 - b3 = 0 a = b，也就是 a-b = 0

所以分解时最自然先尝试的一次因子是 a-b。这只是候选因子；它是否合适，还要靠乘回去检查。

拿出一次因子 a-b 后，剩下的因子应是二次式。要乘出首项 a3，二次因子的首项应是 a2；要乘出末项 -b3，由于前面已有 -b，二次因子的末项应是 b2。中间项还不知道，先写成 kab。

a3-b3 = (a-b)(a2 + kab + b2) (a-b)(a2 + kab + b2) = a3 + ka2b + ab2 - a2b - kab2 - b3 = a3 + (k-1)a2b + (1-k)ab2 - b3

左边只有 a3 和 -b3，所以混合项必须消失：

k - 1 = 0 k = 1 a3-b3 = (a-b)(a2 + ab + b2)
二次因子的三项都是正的，是因为这样乘回去时两类混合项正好抵消。
03抵消模型

把乘开后的混合项拖到一起

右边乘开后同样会出现六个体积项。中间两对混合项一正一负，拖动它们抵消后，只剩 a3-b3。

04例子检查

分解 8x³ - 125

先写成立方差：

8x3 - 125 = (2x)3 - 53 = (2x-5)((2x)2 + (2x)·5 + 52) = (2x-5)(4x2 + 10x + 25)

检查时要特别注意：(2x)2=4x2，不是 2x2。
