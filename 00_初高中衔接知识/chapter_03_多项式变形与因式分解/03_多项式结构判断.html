多项式结构判断

结构判断不是给式子贴标签，而是判断这个式子适合用什么方式看。不同写法会暴露不同信息，后面的解方程、看零点、判断正负和化简分式都可能依赖它。

01判断目的

先看结构，是为了看见隐藏信息

学因式分解时，真正困难的地方往往不是某一个公式，而是看到一个多项式时，不知道应该从哪里下手。结构判断要解决的就是这个问题。

同一个式子，不同写法看见的信息不同。

x²-5x+6 → 展开形式：看出它由哪些项组成。
(x-2)(x-3) → 乘积形式：直接看出零点 2 和 3。

所以结构判断不是为了回答“这叫什么结构”，而是在问：这个式子适合怎样改写，才能完成当前问题？

解方程

把加减形式改成乘积形式，才能用“乘积为 0”来找解。

看零点

函数写成因式相乘，和 x 轴的交点会更容易看出来。

判断正负

乘积形式可以把符号问题拆成几个因式的正负问题。

化简分式

先分解，再看公共因式；同时不能忘记原来的限制条件。

题目不会只问结构，但下一步常常依赖结构。

x²+7x+12=0 → 看出二次三项式 → (x+3)(x+4)=0 → 解方程
(x²-9)/(x-3) → 看出平方差 → (x-3)(x+3)/(x-3) → 看清约分
(x+1)²-5(x+1)+6 → 看出重复整体 → 先保留 x+1，不急着展开

常见结构信号

共同部分6x²-9x

每一项都有共同因子，先提公因式。

平方差x²-25

两项相减，且都能写成平方。

二次三项式x²+7x+12

检查乘法来源，和与积要同时匹配。

完全平方x²+10x+25

检查是否满足 A²±2AB+B²。

重复整体(x²+x)²+3(x²+x)+2

先把重复整体临时看成一个对象。

四项分组ax+ay+bx+by

分组后要制造同一个括号。

02判断顺序

按结构选择方法

判断顺序不是把方法名称背一遍，而是用几个问题逐步缩小范围。

01 每一项都有共同部分吗？ → 先提公因式
02 两项相减，且都能写成平方吗？ → 看平方差
03 三项像平方展开吗？ → 看完全平方
04 二次三项式能看回两个一次因式吗？ → 十字相乘或配方
05 有没有重复出现的整体？ → 先保留整体，必要时换元
06 四项或更多能不能制造共同括号？ → 尝试分组
03常见错误

少检查一步，就会分解错误

错误：只看常数项，把 x²+5x+6 写成 (x+1)(x+6)。

(x+1)(x+6)=x²+7x+6

常数项对了，但一次项错了，所以分解错了。

04例题检查

先说结构，再写步骤

例 1：x²-x-12 是二次三项式，检查两个数的和与积。

3+(-4)=-1，3·(-4)=-12
x²-x-12=(x+3)(x-4)

例 2：(x-2)²-4(x-2)+3 有重复整体。

设 t=x-2，得到 t²-4t+3=(t-1)(t-3)
换回去：(x-2)²-4(x-2)+3=(x-3)(x-5)
05练习要求

先写结构判断

6x²-9x x²-25 x²+10x+25 xy+3x+2y+6 (x+1)²-5(x+1)+6

方法只是结构信号的回应。真正要练的是：先看这个多项式像从哪一种乘法结构展开来，最后再乘回去检查。
