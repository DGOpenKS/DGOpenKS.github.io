# 小结

这一章只围绕一件事：解集追踪。

方程问：

> 哪些数让关系相等？

不等式问：

> 哪些范围让关系成立？

约束问：

> 哪些数从一开始就不能进入讨论？

所以，计算出来的结果常常只是候选结果。真正可靠的答案，要经过完整检查：

```text
先确定允许范围
→ 再判断变形是否保持解集
→ 再求出候选结果
→ 最后把候选结果放回原来的条件里检查
→ 写出最终解集
```

方程变形时，要注意平方、约分、乘以含未知数的式子可能改变解集。

分式方程和根式方程，要先写原式约束，再求候选解，最后验回原式。

不等式变形时，要先判断乘除对象的符号。

二次不等式要用零点分区间。

分式不等式要把分子零点和分母禁点一起放到数轴上。

最后真正要记住的是：计算出来的结果只是候选结果，只有通过原条件检查的，才是真正的解。
