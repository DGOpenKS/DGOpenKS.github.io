Chapter 05

解集追踪

这一章承接表达式条件，继续讨论方程和不等式：哪些数能真正留下来成为解，哪些数会被约束或验根筛掉。

01核心

解不是算完就结束，而是被筛选后留下来的数

方程问哪些数让关系相等，不等式问哪些范围让关系成立，约束负责排除一开始就不能进入讨论的数。

先确定允许范围 再判断变形是否保持解集 再求出候选结果 最后把候选结果放回原式检查
02目录

按解集追踪的顺序阅读

00解集追踪

先把解看成被条件留下来的点或范围。

点解 / 区间解 / 约束
 01方程变形

看哪些变形保持解集，哪些变形会产生增根。

{2} → {-2, 2}
 02分式根式方程

先看约束，再平方或去分母，最后验根。

约束 → 候选 → 验根
 03不等式变形

乘除前先判断符号，负数会让方向翻转。

-3x ≥ 6 → x ≤ -2
 04二次不等式

零点不是答案，而是分区间的边界。

(x-1)(x-3)>0
 05分式不等式

分子零点和分母禁点一起分区间。

零点可取，禁点必排除
 06小结

把允许范围、候选结果和验回原式合成一条检查线。

候选结果 ≠ 最终解
