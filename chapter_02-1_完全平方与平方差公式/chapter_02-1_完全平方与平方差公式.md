# 02-1 完全平方与平方差公式：怎样从面积结构看懂乘法公式

> 本文档是 DGOpenKS 的核心教学 Markdown。
> 它不是普通文章，不是页面实现说明，也不是视频脚本。
> 它的任务是设计清楚一个数学内容的学习逻辑，让使用者更容易真正理解数学。

## 1. 本节核心问题

本节只解决一个问题：

学生怎样通过面积结构理解完全平方公式和平方差公式，并能判断什么时候展开、什么时候反向分解。

本节只讨论：

```text
(a+b)^2
(a-b)^2
a^2-b^2
```

不扩展到立方公式，也不展开讲一般因式分解方法。

## 2. 学生容易误解的地方

学生最容易把平方“分配”到括号里的每一项：

```text
(a+b)^2 = a^2+b^2
```

这个误解来自对指数的表面理解：学生看见平方，就以为每一项分别平方即可，却没有看见 `(a+b)^2` 实际上是 `(a+b)(a+b)`。

第二个误解是把完全平方和平方差混在一起：

```text
(a-b)^2 误看成 a^2-b^2
```

这通常是因为学生只看见符号里有减号，却没有区分“差的平方”和“平方的差”。

第三个误解是反向识别时只看首尾平方，不检查中间项。例如把：

```text
x^2+8x+25
```

误认为完全平方。学生看见 `x^2` 和 `25` 都是平方，却忘了中间项必须是两端平方根乘积的两倍。

## 3. 正确理解的关键结构

关键结构是面积。

`(a+b)^2` 表示边长为 `a+b` 的正方形面积。把边长拆成 `a` 和 `b` 后，整个正方形由四块组成：

```text
a^2, ab, ab, b^2
```

所以中间项不是一个 `ab`，而是两个 `ab`：

```text
(a+b)^2 = a^2+2ab+b^2
```

`(a-b)^2` 表示边长为 `a-b` 的正方形面积。可以从边长为 `a` 的大正方形出发，拿走两条面积为 `ab` 的长方形，但角落的 `b^2` 被拿走了两次，需要加回来：

```text
(a-b)^2 = a^2-2ab+b^2
```

`a^2-b^2` 表示大正方形挖掉小正方形后的剩余面积。剩余部分可以重排成长方形，长和宽分别是：

```text
a+b
a-b
```

因此：

```text
a^2-b^2 = (a+b)(a-b)
```

## 4. 学习路径

1. 先区分三个对象：`(a+b)^2`、`(a-b)^2`、`a^2-b^2`，尤其区分“差的平方”和“平方的差”。

2. 用边长为 `a+b` 的正方形解释 `(a+b)^2`，让学生看见两个 `ab` 的来源。

3. 用从 `a^2` 中拿走两条 `ab`、再加回重叠 `b^2` 的结构解释 `(a-b)^2`。

4. 用大正方形挖掉小正方形再重排的结构解释 `a^2-b^2`。

5. 让学生练习反向识别：三项完全平方必须检查中间项；两项相减且都是平方才适合平方差。

6. 最后用乘回去检查公式是否成立，建立“公式可以验证”的意识。

## 6. 必要的图像或交互

需要一个 `(a+b)^2` 面积拆分图，帮助学生看见完整正方形由四块组成：

```text
a^2, ab, ab, b^2
```

这个图像的教学任务是解释为什么中间项是 `2ab`。

需要一个 `(a-b)^2` 面积图，帮助学生看见从 `a^2` 中减去两条 `ab` 后，重叠的 `b^2` 必须加回来。

这个图像的教学任务是解释为什么公式最后仍然有 `+b^2`。

需要一个 `a^2-b^2` 重排图，帮助学生看见“大正方形挖掉小正方形”可以变成长方形：

```text
(a+b)(a-b)
```

这个图像的教学任务是解释平方差为什么能因式分解。

交互只需要支持改变 `a`、`b` 并观察面积结构同步变化。重点不是操作本身，而是让学生确认公式描述的是同一块面积的不同看法。

## 7. 练习与反馈

基础练习：

- 展开 `(x+3)^2`。
- 展开 `(x-4)^2`。
- 分解 `x^2-25`。

反馈重点：

- 是否写出中间项 `2ab`。
- 是否区分了 `(x-4)^2` 和 `x^2-16`。
- 是否识别出平方差是两个平方相减。

变式练习：

- 展开 `(2x+1)^2`。
- 识别 `x^2+10x+25` 是否为完全平方。
- 分解 `9x^2-16`。

反馈重点：

- 是否把 `2x` 作为整体平方根。
- 是否检查中间项等于 `2·x·5`。
- 是否把 `9x^2` 识别为 `(3x)^2`。

迁移练习：

- 判断 `x^2+6x+9`、`x^2+6x+8`、`x^2+9` 哪些能用本节公式直接处理。

反馈重点：

- `x^2+6x+9` 是完全平方。
- `x^2+6x+8` 不是完全平方，但可以用其他因式分解方法。
- `x^2+9` 不是平方差，因为不是两个平方相减。

常见错误反馈：

- 如果学生写 `(a+b)^2=a^2+b^2`，反馈应指向“中间还有两个 `ab` 面积块”。
- 如果学生把 `(a-b)^2` 写成 `a^2-b^2`，反馈应要求区分“差的平方”和“平方的差”。
- 如果学生误用平方差处理 `a^2+b^2`，反馈应强调平方差必须是“两个平方相减”。
- 如果学生反向识别完全平方时只看首尾平方，反馈应要求检查中间项是否为两端平方根乘积的两倍。

## 8. 学生最后应该带走的理解

完全平方和平方差公式不是符号模板，而是同一块面积在“拆分、补回、重排”中的不同表达。
