# 02-1 完全平方与平方差公式：先看面积，再写公式

## 1. 先看懂这件事

完全平方公式和平方差公式，表面上是几条代数公式，真正要理解的是同一件事：

> 一个面积可以怎样被拆开、补回或重新摆放。

如果只把它们当成要背的式子，很容易记混；如果先看清它们对应的面积对象，公式会自然出现，错误也更容易被发现。

这一节只解决一个核心问题：看到含平方的二次式时，怎样先判断它表示的面积结构，再决定能不能使用完全平方或平方差。

## 2. 先从一块正方形面积看起

先看一个边长为 `a+b` 的正方形。

这个正方形的边不是一整段单独的长度，而是由 `a` 和 `b` 两段拼成。沿着横向和纵向各切一次，里面会出现四块面积：

```text
a^2, ab, ab, b^2
```

这里最重要的观察是：中间不是一块 `ab`，而是两块 `ab`。

所以当你看到 `(a+b)^2` 时，不要急着把 `a` 和 `b` 分别平方。先把它读成：

```text
(a+b)(a+b)
```

两个括号相乘时，`a` 和 `b` 会交叉出现两次：

```text
a·b
b·a
```

这就是中间项 `2ab` 的来源。它不是额外背出来的规则，而是面积被切开以后自然出现的两块长方形。

这里适合配一张面积图：重点不是装饰，而是让你看见 `a^2`、`b^2` 和两个 `ab` 分别从哪里来。

## 3. 这里最容易看错什么

第一个易错点，是把 `(a+b)^2` 写成：

```text
a^2+b^2
```

这个错误的根源是把“整体平方”看成了“分别平方”。检查方法很简单：只要看到括号平方，先把它改写成两个相同括号相乘，再检查有没有漏掉交叉项。

第二个易错点，是把 `(a-b)^2` 和 `a^2-b^2` 混在一起。

它们都含有减号和平方，但对象不同：

```text
(a-b)^2  是差的平方
a^2-b^2  是平方的差
```

前者先把 `a-b` 看成一个整体，再平方；后者是两个平方面积已经形成以后，再做相减。只要对象不同，公式就不能混用。

第三个易错点，是看到首尾都是平方，就立刻判断为完全平方。

例如：

```text
x^2+8x+25
```

首尾确实是 `x^2` 和 `5^2`，但完全平方还要检查中间项。这里如果是完全平方，中间项应该是：

```text
2·x·5 = 10x
```

不是 `8x`，所以它不是完全平方。

第四个易错点，是看到两个平方就想用平方差。

平方差的关键不是“有两个平方”，而是“两个平方相减”。所以 `a^2+b^2` 不能直接写成 `(a+b)(a-b)`，因为它不是差。

## 4. 顺着面积推出公式

现在再把三个公式写出来，它们就不是孤立的记忆内容了。

### 和的平方：拆开面积

边长为 `a+b` 的正方形，被切成四块：

```text
a^2, ab, ab, b^2
```

所以同一块面积可以写成：

```text
(a+b)^2 = a^2+2ab+b^2
```

检查时记住一句话：和的平方不是分别平方，中间还有两块交叉面积。

### 差的平方：减去以后要补回

再看 `(a-b)^2`。

可以从边长为 `a` 的大正方形出发。为了得到边长为 `a-b` 的小正方形，我们会减去两条宽为 `b` 的长方形。每条长方形面积都是 `ab`。

但这两条长方形在角落重叠了一块 `b^2`，这块角落被减了两次。为了得到真正剩下的面积，必须把它补回来。

所以：

```text
(a-b)^2 = a^2-2ab+b^2
```

检查时记住一句话：括号里是减号，不代表最后一项也是减号；最后的 `+b^2` 是补回多减的角落。

### 平方差：剩下的面积可以重排

最后看 `a^2-b^2`。

它表示的是：

```text
边长为 a 的正方形面积 - 边长为 b 的正方形面积
```

拿掉小正方形后，剩下的部分可以重新摆成一个长方形。这个长方形的一条边是 `a-b`，另一条边是 `a+b`。

所以：

```text
a^2-b^2 = (a+b)(a-b)
```

检查时记住一句话：平方差必须是两个平方相减，剩下的面积才能重排成长方形。

## 5. 用例子检查理解

例 1：

```text
(x+3)^2
```

先读成两个相同括号相乘：

```text
(x+3)(x+3)
```

你会看到两次交叉项：

```text
x·3
3·x
```

所以：

```text
(x+3)^2 = x^2+6x+9
```

例 2：

```text
(x-4)^2
```

它是“差的平方”，不是“平方的差”。所以不能写成 `x^2-16`。

正确结构是：

```text
(x-4)^2 = x^2-8x+16
```

最后的 `+16` 对应补回的角落。

例 3：

```text
x^2-25
```

先把它看成两个平方相减：

```text
x^2-5^2
```

所以可以使用平方差：

```text
x^2-25 = (x+5)(x-5)
```

例 4：

```text
x^2+8x+25
```

它看起来像完全平方，但检查中间项会发现：

```text
2·x·5 = 10x
```

中间项不是 `10x`，所以它不能写成 `(x+5)^2`。

## 6. 换几个练习再判断

练习时先判断结构，再动手展开或分解。

基础练习可以换数生成：

```text
(x+n)^2
(x-n)^2
x^2-n^2
```

其中 `n` 可以换成小的正整数。检查重点是：前两类有没有写出中间项，第三类是不是两个平方相减。

变式练习可以换成：

```text
(mx+n)^2
(mx-n)^2
m^2x^2-n^2
```

其中 `m,n` 可以换成小的正整数。检查重点是：平方根不一定是 `x` 和 `n`，也可能是 `mx` 和 `n`。

判断练习要故意放入“看起来像但不能用”的式子，例如：

```text
x^2+16
x^2+8x+25
4x^2-9
4x^2+12x+9
```

先不要急着套公式。每一题先问：

```text
它是不是整体平方？
它是不是两个平方相减？
中间项是否匹配？
```

如果答不上来，就回到前面的面积结构，而不是继续硬背公式。

## 7. 以后会在哪里用到

完全平方和平方差以后会反复出现。

在因式分解里，它们帮助你从多项式中看出乘法结构。

在配方里，完全平方能把类似 `x^2+px` 的部分补成一个完整平方，从而读出二次函数的顶点、最值和对称轴。

在方程和不等式里，平方差能把二次表达式拆成两个一次因式，让零点和符号变化更清楚。

所以这节真正练的是一种代数眼光：改变表达式的形式，让隐藏的结构显现出来。

## 8. 最后真正要记住的是

完全平方和平方差公式不是凭空要背的形状；它们来自面积的拆分、补回和重排。只要先看清对象，再检查结构，公式就会变得有来源，也不容易用错。
