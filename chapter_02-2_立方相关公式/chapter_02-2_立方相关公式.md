# 02-2 立方相关公式：为什么立方和、立方差能分解

> 本文档是 DGOpenKS 的核心教学 Markdown。
> 它不是普通文章，不是页面实现说明，也不是视频脚本。
> 它的任务是设计清楚一个数学内容的学习逻辑，让使用者更容易真正理解数学。

## 1. 本节核心问题

本节只解决一个问题：

学生如何理解立方相关公式不是“硬背出来的格式”，而是由三次结构中的选择、补项、切割和抵消决定的。

本节包含三个公式，但它们服务于同一条学习主线：

- `(a+b)^3`：从整体立方体看展开后为什么有四类项。
- `a^3+b^3`：从补项和抵消看为什么能提取 `a+b`。
- `a^3-b^3`：从移除小立方体后的剩余结构看为什么能提取 `a-b`。

本节不继续扩展到更一般的多项式展开，也不处理高次因式分解技巧。

## 2. 学生容易误解的地方

学生常把三次公式当成平方公式的机械升级。

最典型的误解是：

```text
(a+b)^3 = a^3+b^3
```

这个错误来自一种直觉：学生只看见“把外面的 3 分配给括号里的两项”，却没有看见 `(a+b)^3` 实际上是三个括号相乘。

第二个误解是混淆立方和与立方差的二次因子：

```text
a^3+b^3 误写成 (a+b)(a^2+ab+b^2)
a^3-b^3 误写成 (a-b)(a^2-ab+b^2)
```

这个错误通常不是计算能力不足，而是学生只记住了“首平方、尾平方、中间有 ab”，没有理解中间项的符号要服务于乘回去时的抵消。

第三个误解是把立方和公式理解成真实的正体积拼接。事实上 `a^2-ab+b^2` 中含有负项，不能简单解释成三个普通正面积的组合。

## 3. 正确理解的关键结构

关键结构有三个。

第一，`(a+b)^3` 的本质是：

```text
(a+b)(a+b)(a+b)
```

展开时，每一项来自三个括号中的三次选择。因此中间两项的系数 `3` 表示选择位置的数量，而不是额外背诵的数字。

第二，立方和的关键不是直接切拼，而是补项后制造共同因子：

```text
a^3+b^3
=a^2(a+b)-ab(a+b)+b^2(a+b)
```

这里的 `-ab(a+b)` 是有符号量。它帮助学生看见共同因子 `a+b`，但不能被误解成普通正体积。

第三，立方差的关键是剩余体的共同厚度。移除边长为 `b` 的小立方体后，剩余部分可以看成三块都带有共同厚度 `a-b` 的结构：

```text
a^2(a-b)+ab(a-b)+b^2(a-b)
```

因此可以提取：

```text
(a-b)(a^2+ab+b^2)
```

## 4. 学习路径

1. 先让学生判断变形方向：`(a+b)^3` 是展开，`a^3±b^3` 是分解。

2. 用三个括号的选择解释 `(a+b)^3`，让学生看见 `1,3,3,1` 来自选择次数。

3. 用“乘回去检查”确认立方和、立方差的二次因子符号，而不是先背口诀。

4. 对立方和，引入补项与抵消，强调负项是代数结构，不是普通实体拼接。

5. 对立方差，引入“从大立方中移除小立方”的结构，观察剩余部分的共同厚度 `a-b`。

6. 最后比较两个分解公式：一次因子与原式同号；立方和二次因子中间为负；立方差二次因子三项全正。

## 6. 必要的图像或交互

需要一个二项式立方图像，帮助学生看见边长 `a+b` 的立方体能分成四类体块：

```text
a^3, 3a^2b, 3ab^2, b^3
```

这个图像的教学任务是解释系数 `1,3,3,1` 的来源。

需要一个立方和的有符号结构图，帮助学生看见：

```text
a^2(a+b), -ab(a+b), b^2(a+b)
```

这个图像的教学任务是解释共同因子 `a+b`，同时避免学生把负项误解成真实正体积。

需要一个立方差的切割图，帮助学生看见从 `a^3` 中移除 `b^3` 后，剩余结构含有共同厚度：

```text
a-b
```

这个图像的教学任务是解释为什么可以提取 `a-b`。

交互只需要支持改变 `a`、`b` 并同步观察结构变化。重点不是操作复杂度，而是让学生确认结构始终不变。

## 7. 练习与反馈

练习应随机生成，但题型必须始终围绕本节核心知识点：二项式立方展开、立方和分解、立方差分解，以及判断“能不能直接使用这些公式”。

基础练习题族：

- 随机生成 `(x+n)^3` 或 `(mx+n)^3`，要求展开。
- 随机生成 `x^3+n^3` 或 `(mx)^3+n^3`，要求分解立方和。
- 随机生成 `x^3-n^3` 或 `(mx)^3-n^3`，要求分解立方差。

反馈重点：

- 是否先识别了结构类型。
- 是否漏掉 `3a^2b` 或 `3ab^2`。
- 是否把立方和、立方差的二次因子符号写反。

变式练习题族：

- 随机改变一次项系数，例如 `(2x+3)^3`。
- 随机改变常数立方，例如 `8x^3+125`。
- 随机生成需要把整体识别为立方的题，例如 `(2x)^3-4^3`。

反馈重点：

- 是否正确识别 `(2x)^3`、`(3x)^3` 这类整体立方。
- 是否把整个单项式作为 `a`，而不是只看字母部分。

迁移练习题族：

- 随机生成三类判断题：立方和、立方差、都不是。
- “都不是”应包含干扰项，例如 `x^3+9`、`x^3+x^2`、`x^3+2x+1`。

反馈重点：

- 不是所有“三次项加常数”都能直接套立方和。
- 常数也必须是某个数的三次方。
- 如果不是两个立方的和或差，就不能直接使用本节两个分解公式。

常见错误反馈：

- 如果学生写出 `(a+b)^3=a^3+b^3`，反馈应指向“三个括号相乘时还有混合选择”。
- 如果学生写错立方和的二次因子，反馈应要求乘回去检查交叉项是否抵消。
- 如果学生写错立方差的二次因子，反馈应提醒“差号已经在一次因子里，二次因子负责制造抵消”。
- 如果学生把不能直接套公式的题误判为立方和或立方差，反馈应要求先识别两端是否都是完整立方。

## 8. 学生最后应该带走的理解

立方相关公式的核心不是背符号，而是看见三次结构中哪些项来自选择，哪些项为了制造共同因子，哪些项会在乘回去时抵消。
