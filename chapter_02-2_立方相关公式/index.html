1三次选择

先从三个括号相乘看起

把 (a+b)3 读成 (a+b)(a+b)(a+b)。每个括号里选一次，三个括号的选择合在一起，就产生了四类项。

(a+b)3 = a3 + 3a2b + 3ab2 + b3
1 块 a3 3 块 a2b 3 块 ab2 1 块 b3

先把它看成一个整体。 边长为 a+b 的立方体，对应 (a+b)3。
(a+b)3 = a3 + 3a2b + 3ab2 + b3
a3全选 a
3a2b两个 a 一个 b
3ab2一个 a 两个 b
b3全选 b
中间的两个 3 不是额外背出来的数字，而是选择位置的数量。
2易错结构

这里最容易看错什么

三次公式的错误，多数不是算错，而是对象看错。先判断对象，再决定公式。

把整体立方看成分别立方

(a+b)3 是三个相同括号相乘，不是 a3+b3。检查时先写成 (a+b)(a+b)(a+b)。

把差的立方看成立方的差

(a-b)3 是一个整体被立方；a3-b3 是两个立方相减。它们的变形方向不同。

把二次因子的符号写反

立方和的二次因子中间项是 -ab；立方差的二次因子三项全正。符号要服务于乘回去时的抵消。

看到三次项就套公式

在当前整式因式分解范围内，x3+9 不能直接看成立方和。两端都要是完整立方。

对象完整大立方体 a³。

第一类混合项0

当 k=-1 时，两个混合项都抵消，剩下 a³+b³。
(a+b)(a²+kab+b²)
= a³ + (k+1)a²b + (k+1)ab² + b³

4例子检查

用例子检查理解

例 1：展开和的立方
(x+2)3 = x3 + 6x2 + 12x + 8

四类项是 x3、3·x2·2、3·x·22、23。

例 2：展开差的立方
(x-2)3 = x3 - 6x2 + 12x - 8

把它看成 (x+(-2))3。符号不是全负，而是负、正、负交替出现。

例 3：分解立方和
x3 + 27 = (x+3)(x2-3x+9)

先识别 27=33。立方和的二次因子中间项是负号。

例 4：分解立方差
8x3 - 125 = (2x-5)(4x2+10x+25)

这里 8x3=(2x)3。整个 2x 才是被立方的对象。

反例：x3+9 看起来像立方和，但在当前整式因式分解范围内，9 不能作为完整三次方来处理。

6带走理解

最后真正要记住的是

立方相关公式不是一串容易混淆的符号。整体立方的中间项来自三个括号的混合选择；立方和、立方差的二次因子来自让混合项抵消。

整体立方

先写成三个括号相乘，再数选择位置。

立方和

一次因子同号，二次因子中间项为负。

立方差

一次因子同号，二次因子三项全正。

不能直接用

两端不是完整立方时，先停下来判断。
