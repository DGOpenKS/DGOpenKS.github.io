# 第二章：代数变形与表达式结构

## 0. 章节定位

本章不是“初中公式复习”，而是高中数学的计算与结构训练。

高中数学里，很多题目表面上考函数、方程、不等式、导数、数列、解析几何，真正卡住学生的地方却往往是：

- 看不出式子结构
- 不知道该展开还是分解
- 配方不熟
- 分式化简丢条件
- 根式变形不会处理
- 绝对值只会机械去符号
- 不会把复杂整体换元
- 不会为了考试目标选择合适的变形

所以，本章的核心不是“会算几个式子”，而是建立一种更强的能力：

> 看懂表达式的结构，并把它变成最适合解决问题的形态。

本章核心句：

> 同一个式子，换一种写法，就换了一种看法。

---

## 1. 本章学习目标

学完本章，学生应该做到：

1. 能区分表达式中的项、因子、整体结构。
2. 能熟练使用完全平方、平方差、立方和差等乘法公式。
3. 能解释展开、因式分解、配方分别暴露什么信息。
4. 能掌握高中常用因式分解方法：提公因式、公式法、十字相乘、分组、换元、配方。
5. 能正确处理分式化简，并保留限制条件。
6. 能进行根式化简、分母有理化和常见嵌套根号拆解。
7. 能从距离角度理解绝对值，并能进行分段讨论。
8. 能根据考试目标选择代数变形路线。
9. 能说明本章工具如何服务后续方程、不等式、函数图像、坐标几何。

3. **怎么做**：遇到题目时具体怎样操作。

### 3.1 公式渲染规范

页面中的公式必须统一渲染风格。

要求：

- 二次方统一使用 MathJax：`$x^2$`，不要混用 `x²`、普通文本和图片。

- 例题推导使用分步公式，不把多步变形挤在一行。



## 5. 看结构，做变形

### 5.1 它解决什么问题

高中代数的第一步不是立刻计算，而是判断：

```text
这个式子现在是什么结构？
我希望它变成什么结构？
变形后能暴露什么信息？
```

结构判断和变形动作不能分开讲。因为“看见结构”只是开始，真正服务考试的是能立刻选择合适的变形路线。

### 5.2 四类常见结构

| 当前结构 | 结构信号 | 典型例子 | 合适动作 | 暴露的信息 |
|---|---|---|---|---|
| 项结构 | 加减连接，存在同类项 | `3x^2-2x+5x^2+7x` | 合并同类项 | 标准形式、系数 |
| 乘法结构 | 括号相乘 | `(x+2)(x+5)` | 展开并合并 | 二次项、一次项、常数项 |
| 公因式结构 | 每一项含同一因子 | `4x(x-2)+3(x-2)` | 提公因式 | 乘法结构、零点线索 |
| 整体结构 | 复杂部分重复出现 | `(x^2+x)^2-5(x^2+x)+6` | 整体代换 | 把复杂式降成熟悉二次式 |

### 5.3 展开：乘法结构变加法结构

展开的本质是分配律：

```text
a(b+c)=ab+ac
```

例：

```text
(x+2)(x+5)
=x^2+5x+2x+10
=x^2+7x+10
```

展开适合用于：

- 整理标准形式
- 比较系数
- 与函数一般式、方程一般式对接

易错点：两个括号相乘时，中间项来自两次交叉乘积，不能只乘首项和末项。

### 5.4 合并同类项：相同结构的数量相加

同类项不是“看起来都有 x 就能合并”，而是字母部分完全相同。

例：

```text
3x^2-2x+5x^2+7x
=8x^2+5x
```

本质是：

```text
3x^2+5x^2=(3+5)x^2
```

合并同类项适合用于把式子压缩成标准形式，为后面的因式分解、配方或函数分析做准备。

### 5.5 提公因式：加法结构变乘法结构

提公因式的本质是反向分配律：

```text
ax+ay=a(x+y)
```

高中更重要的是识别“整体公因式”。

例：

```text
4x(x-2)+3(x-2)
=(4x+3)(x-2)
```

这里共同的不是单个字母，而是整个括号：

```text
x-2
```

提公因式适合用于：

- 因式分解
- 求零点
- 判断符号
- 化简分式前制造可约因子

### 5.6 整体代换：把复杂部分先看成一个新对象

当同一块结构重复出现时，不要急着展开。先把它命名为一个临时变量。

例：

```text
(x^2+x)^2-5(x^2+x)+6
```

第一步，识别重复整体：

```text
x^2+x
```

第二步，令：

```text
t=x^2+x
```

原式变成：

```text
t^2-5t+6
```

第三步，按熟悉的二次式分解：

```text
t^2-5t+6=(t-2)(t-3)
```

第四步，换回原变量：

```text
(x^2+x-2)(x^2+x-3)
```

第五步，继续检查能否分解：

```text
x^2+x-2=(x+2)(x-1)
```

所以：

```text
(x^2+x)^2-5(x^2+x)+6
=(x+2)(x-1)(x^2+x-3)
```

整体代换常出现在：

- 高次多项式分解
- 复合函数表达式
- 数列递推式
- 三角表达式预备
- 导数符号判断前的化简

### 5.7 本模块例题设计

HTML 中的练习不应是孤立地“判断项、因子、整体”，而应设计成自然的变形任务。

例题库：

```text
请将 (x+2)(x+5) 展开并合并同类项。
请把 4x(x-2)+3(x-2) 因式分解。
请观察 (x^2+x)^2-5(x^2+x)+6，找出可整体处理的部分并完成变形。
请先判断结构，再化简 3x^2-2x+5x^2+7x。
```

每道题的反馈必须包含：

- 当前结构
- 选择的变形动作
- 关键步骤
- 变形后暴露的信息
- 易错提醒

### 5.8 常见错误

错误一：看到复杂式子就展开，忽略整体结构。

错误二：把同类项理解成“都有 x”，导致把 `x^2` 和 `x` 合并。

错误三：提公因式时只找单个字母，不找整体括号。

错误四：整体代换后忘记换回原变量。

错误五：换回后不继续检查能否再次分解。

## 6. 乘法公式

### 6.1 它解决什么问题

乘法公式有两个方向：

```text
乘积 → 展开
展开 → 分解
```

高中更重要的是第二个方向：从复杂式子中识别结构。

### 6.2 完全平方公式

公式：

```text
(a+b)^2 = a^2 + 2ab + b^2
```

```text
(a-b)^2 = a^2 - 2ab + b^2
```

反向识别：

```text
a^2 + 2ab + b^2 = (a+b)^2
```

```text
a^2 - 2ab + b^2 = (a-b)^2
```

### 6.3 图解解释

边长为 `a+b` 的正方形可以分成四块：

```text
a^2, ab, ab, b^2
```

所以总面积是：

```text
a^2 + 2ab + b^2
```

这解释了为什么中间项是 `2ab`。

### 6.4 例题

展开：

```text
(2x-3)^2
```

解：

```text
(2x-3)^2
= (2x)^2 - 2·2x·3 + 3^2
= 4x^2 - 12x + 9
```

反向识别：

```text
9x^2 + 12x + 4
```

解：

```text
9x^2 + 12x + 4
= (3x)^2 + 2·3x·2 + 2^2
= (3x+2)^2
```

### 6.5 平方差公式

公式：

```text
(a+b)(a-b)=a^2-b^2
```

反向：

```text
a^2-b^2=(a+b)(a-b)
```

例题：

```text
4x^2 - 25
= (2x)^2 - 5^2
= (2x+5)(2x-5)
```

### 6.6 立方和与立方差

公式：

```text
a^3+b^3=(a+b)(a^2-ab+b^2)
```

```text
a^3-b^3=(a-b)(a^2+ab+b^2)
```

例题：

```text
x^3-8
= x^3-2^3
= (x-2)(x^2+2x+4)
```

```text
8x^3+27
= (2x)^3+3^3
= (2x+3)(4x^2-6x+9)
```

### 6.7 每个公式的图解要求

HTML 中不能只展示公式。至少需要以下图解：

#### 完全平方公式图

交互要求：

- 滑块控制 `a` 与 `b`
- 正方形面积随 `a:b` 比例变化
- 四个区域分别标注 `$a^2$`、`$ab$`、`$ab$`、`$b^2$`
- 公式同步显示：

```text
(a+b)^2 = a^2 + 2ab + b^2
```

学习目标：

学生必须看到 `2ab` 来自两个长方形。

#### 平方差公式图

交互要求：

- 滑块控制 `a` 与 `b`
- 显示面积 `$a^2-b^2$`
- 通过切割/重排，转成面积为 `$(a+b)(a-b)$` 的长方形
- 可以用“播放重排”按钮或分步状态切换

学习目标：

学生看到：

```text
a^2-b^2=(a+b)(a-b)
```

不是代数巧合，而是面积守恒。

#### 立方和差公式图

立方公式不一定要做复杂 3D，但需要结构图：

```text
a^3-b^3
→ (a-b)(a^2+ab+b^2)
```

建议使用“因子分层卡片”展示：

- 第一层：识别 `a^3-b^3`
- 第二层：提取一次因子 `a-b`
- 第三层：剩余二次因子 `a^2+ab+b^2`

### 6.8 随机例题设计

HTML 中应加入“公式识别练习”。

题型：

1. 判断适用哪一个公式。
2. 完成展开。
3. 反向因式分解。

例题库：

```text
(x+4)^2
(2x-3)^2
x^2-16
9x^2-25
x^3-27
8x^3+1
x^2+10x+25
4x^2-12x+9
```

反馈要说明：

- 识别到了什么结构
- 使用哪一个公式
- 为什么中间项或二次因子这样出现

### 6.9 公式识别清单

看到下面结构时，要立即联想到公式：

| 结构 | 可能公式 |
|---|---|
| 两个平方相减 | 平方差 |
| 三项，中间项是两端平方根乘积的 2 倍 | 完全平方 |
| 两个立方相加或相减 | 立方和差 |
| 根式分母形如 `√a+√b` | 共轭与平方差 |

### 6.10 常见错误

错误：

```text
(a+b)^2 = a^2+b^2
```

正确：

```text
(a+b)^2 = a^2+2ab+b^2
```

错误：

```text
a^2+b^2=(a+b)(a-b)
```

平方差只能用于：

```text
a^2-b^2
```

## 7. 因式分解方法库

### 7.1 它解决什么问题

因式分解把加法结构变成乘法结构。

它最常用于：

- 解方程
- 解不等式
- 判断符号
- 找零点
- 化简分式
- 分析函数图像与 x 轴交点

核心思想：

> 乘积为 0，意味着至少一个因子为 0。

### 7.2 为什么因式分解重要

因式分解不只是把式子写短。

它至少有三个核心作用：

1. **求零点**：乘积为零时，至少一个因子为零。
2. **判断符号**：因子正负决定整个乘积正负。
3. **暴露结构**：把隐藏的根、区间、公共结构显出来。

例如：

```text
x^2 - 5x + 6
```

如果保持一般式，零点并不直观。

分解后：

```text
x^2 - 5x + 6 = (x-2)(x-3)
```

可以直接读出：

```text
x=2, x=3
```

并能进一步判断：

```text
(x-2)(x-3)>0
```

的解集。

### 7.3 方法一：提公因式

```text
3x^2 + 6x = 3x(x+2)
```

整体公因式：

```text
a(x+y)+b(x+y)=(a+b)(x+y)
```

### 7.4 方法二：公式法

```text
x^2-9=(x+3)(x-3)
```

```text
x^2+6x+9=(x+3)^2
```

### 7.5 方法三：十字相乘

适用于很多二次三项式。

#### 首项系数为 1

```text
x^2-5x+6
```

找两个数：

```text
和 = -5
积 = 6
```

得到：

```text
-2, -3
```

所以：

```text
x^2-5x+6=(x-2)(x-3)
```

#### 首项系数不为 1

```text
2x^2+5x+3
```

尝试：

```text
(2x+3)(x+1)
```

检查：

```text
(2x+3)(x+1)
=2x^2+2x+3x+3
=2x^2+5x+3
```

### 7.6 方法四：分组分解

```text
ax+ay+bx+by
```

分组：

```text
(ax+ay)+(bx+by)
```

提公因式：

```text
a(x+y)+b(x+y)
```

再提：

```text
(a+b)(x+y)
```

### 7.7 方法五：换元

```text
(x^2+x)^2-5(x^2+x)+6
```

令：

```text
t=x^2+x
```

得到：

```text
t^2-5t+6=(t-2)(t-3)
```

换回：

```text
(x^2+x-2)(x^2+x-3)
```

### 7.8 方法六：配方法

配方不是普通因式分解，但它是二次式结构分析中最重要的方法之一。

```text
x^2+6x+5
```

补成完全平方：

```text
x^2+6x+9-9+5
```

所以：

```text
x^2+6x+5=(x+3)^2-4
```

### 7.9 配方的一般形式

对：

```text
ax^2+bx+c
```

当 `a≠0` 时：

```text
ax^2+bx+c
= a(x + b/(2a))^2 + (4ac-b^2)/(4a)
```

也可以写成：

```text
a(x-h)^2+k
```

其中：

```text
h = -b/(2a)
```

这就是二次函数对称轴公式的来源。

### 7.10 图解与交互要求

因式分解模块必须增加图解，而不是只列方法。

推荐交互：

#### 交互一：一般式与因式分解式切换

以：

```text
y=x^2-5x+6
```

为例。

页面显示：

```text
y=x^2-5x+6
```

点击“分解”后显示：

```text
y=(x-2)(x-3)
```

图像上同步标出零点：

```text
x=2, x=3
```

#### 交互二：数轴符号区间

在图像下方增加数轴：

```text
(-∞,2)   (2,3)   (3,+∞)
```

标出：

```text
(x-2)(x-3)>0
```

在哪些区间成立。

#### 交互三：方法选择练习

随机给出表达式，让学生先选择方法：

```text
提公因式 / 公式法 / 十字相乘 / 分组 / 换元 / 配方
```

选择后显示步骤。

例题库：

```text
3x^2+6x
x^2-9
x^2-5x+6
2x^2+5x+3
ax+ay+bx+by
(x^2+x)^2-5(x^2+x)+6
x^2+6x+5
```

### 7.11 方法选择

| 表达式特征 | 优先方法 |
|---|---|
| 每一项都有共同因子 | 提公因式 |
| 出现平方差、完全平方 | 公式法 |
| 二次三项式 | 十字相乘或求根公式 |
| 四项且能分组 | 分组分解 |
| 重复出现复杂整体 | 换元 |
| 需要最值、顶点、范围 | 配方法 |

## 8. 分式变形

### 8.1 它解决什么问题

分式变形常用于：

- 化简函数表达式
- 求定义域
- 解分式方程
- 解分式不等式
- 处理解析几何中的斜率与距离

分式最重要的原则：

> 分母不能为 0。

### 8.2 约分只能约因子

正确：

```text
x(x+1)/(x+1)=x,  x≠-1
```

错误：

```text
(x+1)/x
```

不能把分子中的 `x` 和分母中的 `x` 约掉，因为分子是和，不是乘积。

### 8.3 化简后要保留条件

```text
(x^2-1)/(x-1)
```

先分解：

```text
x^2-1=(x-1)(x+1)
```

所以：

```text
(x^2-1)/(x-1)=x+1
```

但原式要求：

```text
x≠1
```

完整结论：

```text
(x^2-1)/(x-1)=x+1, x≠1
```

### 8.4 通分

```text
1/x + 1/(x+1)
```

通分：

```text
(x+1)/(x(x+1)) + x/(x(x+1))
```

得到：

```text
(2x+1)/(x(x+1))
```

条件：

```text
x≠0, x≠-1
```

### 8.5 分式方程的增根

例如：

```text
(x+1)/(x-1)=2/(x-1)
```

两边同乘 `x-1` 前必须说明：

```text
x≠1
```

同乘后：

```text
x+1=2
```

所以：

```text
x=1
```

但 `x=1` 使原分母为 0，因此是增根，应舍去。

所以原方程无解。

### 8.6 图解与交互要求

分式模块必须强调“表达式看起来相同，但对象不一定相同”。

推荐交互：

- 左侧显示原式：

```text
f(x)=\frac{x^2-1}{x-1}
```

- 右侧显示化简式：

```text
g(x)=x+1
```

- 图像上在 `x=1` 处挖空
- 说明：

```text
f(x)=g(x), 但 x≠1
```

随机例题应要求学生写出限制条件。

例题库：

```text
(x^2-4)/(x-2)
(x^2-1)/(x+1)
(x^2+3x)/(x)
(x^2-9)/(x^2-3x)
1/x + 1/(x+1)
```

### 8.7 考试提醒

分式题必须写清：

```text
先定条件，再变形，最后验根。
```

## 9. 根式变形

### 9.1 它解决什么问题

根式变形常用于：

- 化简函数表达式
- 求定义域
- 分母有理化
- 处理含根式方程
- 处理复杂代数式估值

根式最重要的限制：

```text
偶次根号内必须非负
```

### 9.2 根式化简

```text
√12 = √(4·3)=2√3
```

```text
√50 = √(25·2)=5√2
```

核心是拆出平方因子。

### 9.3 分母有理化

单项根式：

```text
1/√2 = √2/2
```

二项根式：

```text
1/(√3+1)
```

乘共轭式：

```text
1/(√3+1) · (√3-1)/(√3-1)
```

得到：

```text
(√3-1)/(3-1) = (√3-1)/2
```

这里用到平方差：

```text
(a+b)(a-b)=a^2-b^2
```

### 9.4 嵌套根号拆解

常见结构：

```text
√(a+2√b)
```

尝试写成：

```text
√m+√n
```

平方右边：

```text
(√m+√n)^2=m+n+2√(mn)
```

所以需要：

```text
m+n=a
mn=b
```

例子：

```text
√(7+4√3)
```

先把中间项写成：

```text
4√3 = 2√12
```

所以：

```text
a=7, b=12
```

找：

```text
m+n=7
mn=12
```

得到：

```text
m=4, n=3
```

因此：

```text
√(7+4√3)=√4+√3=2+√3
```

带减号：

```text
√(7-4√3)=2-√3
```

因为：

```text
(2-√3)^2=7-4√3
```

### 9.5 图解与交互要求

根式模块不能只写公式。

推荐交互：

#### 根式化简

让学生拖动或选择平方因子：

```text
√12 = √(4·3) = 2√3
```

#### 分母有理化

用共轭按钮显示：

```text
\frac{1}{\sqrt3+1}
·
\frac{\sqrt3-1}{\sqrt3-1}
```

并高亮平方差：

```text
(\sqrt3+1)(\sqrt3-1)=3-1
```

#### 嵌套根号拆解

显示配对面板：

```text
\sqrt{a+2\sqrt b}
```

需要寻找：

```text
m+n=a
mn=b
```

例题库：

```text
√(7+4√3)
√(7-4√3)
√(13+4√10)
√(13-4√10)
√(5+2√6)
√(5-2√6)
```

### 9.6 根式方程提醒

根式方程常需要平方，但平方可能引入增根。

流程：

```text
定条件
→ 隔离根式
→ 平方
→ 解方程
→ 验根
```

## 10. 绝对值

### 10.1 它解决什么问题

绝对值是距离语言。

它常用于：

- 数轴距离
- 绝对值方程
- 绝对值不等式
- 分段函数
- 函数图像变换

### 10.2 距离意义

```text
|x|
```

表示 `x` 到 `0` 的距离。

```text
|x-a|
```

表示 `x` 到 `a` 的距离。

### 10.3 分段定义

```text
|A| =
A,  A≥0
-A, A<0
```

例如：

```text
|x-3|
```

当：

```text
x≥3
```

有：

```text
|x-3|=x-3
```

当：

```text
x<3
```

有：

```text
|x-3|=3-x
```

### 10.4 绝对值不等式

### 10.5 图解与交互要求

绝对值模块必须使用数轴。

推荐交互：

- 拖动点 `x`
- 固定点 `a`
- 实时显示距离 `$|x-a|$`
- 切换问题：

```text
|x-a|<r
|x-a|>r
```

并在数轴上高亮对应区间。

```text
|x-3|<2
```

表示 `x` 到 `3` 的距离小于 `2`。

所以：

```text
1<x<5
```

```text
|x-3|>2
```

表示 `x` 到 `3` 的距离大于 `2`。

所以：

```text
x<1 或 x>5
```

## 11. 代数变形的解题策略

### 11.1 先判断目标

面对一个式子，先不要急着算。

先问：

```text
我要看见什么？
```

### 11.2 目标与变形方式

| 目标 | 推荐变形 |
|---|---|
| 求零点 | 因式分解 |
| 判断正负 | 因式分解 + 数轴 |
| 求最值 | 配方 |
| 看顶点 | 配方或顶点式 |
| 求定义域 | 检查分母、根号、绝对值 |
| 化简分式 | 因式分解后约分 |
| 处理根式 | 拆平方因子、有理化、嵌套拆解 |
| 处理复杂重复结构 | 换元 |

### 11.3 综合例题一：二次式结构

题目：

```text
求 x^2-6x+5 的最小值。
```

解：

```text
x^2-6x+5
= x^2-6x+9-9+5
= (x-3)^2-4
```

因为：

```text
(x-3)^2≥0
```

所以最小值是：

```text
-4
```

当：

```text
x=3
```

取到。

### 11.4 综合例题二：分式化简与条件

题目：

```text
化简 (x^2-4)/(x-2)
```

解：

```text
(x^2-4)/(x-2)
= ((x-2)(x+2))/(x-2)
= x+2
```

但：

```text
x≠2
```

所以：

```text
(x^2-4)/(x-2)=x+2, x≠2
```

## 12. 常见错误清单

### 12.1 完全平方漏中间项

错误：

```text
(a+b)^2=a^2+b^2
```

正确：

```text
(a+b)^2=a^2+2ab+b^2
```

### 12.2 约分约掉了项

错误：

```text
(x+1)/x = 1+1
```

原因：

`x+1` 是和，不是乘积。

### 12.3 化简后丢条件

错误：

```text
(x^2-1)/(x-1)=x+1
```

完整：

```text
(x^2-1)/(x-1)=x+1, x≠1
```

### 12.4 平方引入增根

根式方程平方后必须验根。

### 12.5 不知道什么时候配方

如果题目问：

- 最值
- 顶点
- 范围
- 圆心半径

通常要考虑配方。

## 13. 与后续章节的联系

### 13.1 连接第三章：方程、不等式与约束

因式分解帮助解方程：

```text
x^2-5x+6=0
```

变成：

```text
(x-2)(x-3)=0
```

因式分解也帮助解不等式：

```text
(x-2)(x-3)>0
```

只要判断两个因子的符号。

### 13.2 连接第四章：函数与图像

配方帮助看二次函数：

```text
y=x^2+6x+5
= (x+3)^2-4
```

直接得到：

- 顶点 `(-3,-4)`
- 对称轴 `x=-3`
- 最小值 `-4`

### 13.3 连接第五章：坐标几何

圆的一般式：

```text
x^2+y^2-4x+6y-12=0
```

配方：

```text
(x-2)^2+(y+3)^2=25
```

得到：

- 圆心 `(2,-3)`
- 半径 `5`

## 14. 可视化与交互设计

本章生成 HTML 时，建议至少安排以下模块。

### 14.1 完全平方面积图

交互：

- 拖动 `a,b`
- 正方形自动分成 `a^2, ab, ab, b^2`

学习目标：

```text
(a+b)^2=a^2+2ab+b^2
```

不是背出来，而是面积拼出来。

### 14.2 平方差切割图

交互：

- 显示 `a^2-b^2`
- 拆下小正方形
- 拼成长方形 `(a+b)(a-b)`

学习目标：

```text
a^2-b^2=(a+b)(a-b)
```

### 14.3 因式分解与零点

交互：

- 一般式与因式分解式切换
- 图像上标出零点

例子：

```text
x^2-5x+6=(x-2)(x-3)
```

### 14.4 配方与顶点

交互：

- 拖动系数
- 一般式实时变成顶点式
- 顶点同步移动

学习目标：

配方不是技巧，是看顶点和最值的方法。

表达式化简后相似，不代表定义域相同。

- 寻找 `m+n=a, mn=b`
- 生成 `√m±√n`

学习目标：

嵌套根号拆解本质是配对。
