1Equation
等式的两种图像

方程的解，是让两边相遇的输入。

天平视角

对等式两边做同样的操作，平衡不变。这解释了移项、同乘、同除。

交点视角

$f(x)=g(x)$ 的解，就是两个图像的交点横坐标。

注意

平方、开方、约分都可能改变条件。根式方程和分式方程要验根。

2Quadratic
二次方程

判别式看根的数量，韦达定理看根的关系。

判别式
$\Delta=b^2-4ac$

大于 0 两根，等于 0 一重根，小于 0 无实根。

韦达定理
$x_1+x_2=-\frac ba,\ x_1x_2=\frac ca$

不直接求根，也能研究根的和与积。

参数意识

参数改变时，根的数量、位置和范围都会跟着改变。

4AM-GM
均值不等式入门

$a+b\ge2\sqrt{ab}$ 是工具，但条件和取等号更重要。

面积 $ab$
$\sqrt{ab}$
当 $a>0,b>0$ 时，$a+b\ge2\sqrt{ab}$

它告诉我们：正数的和与积之间有稳定关系。取等号条件是 $a=b$。

考试用法

看到“正数、和定积最大、积定和最小、取等号”这些信号，可以考虑均值不等式。
