三角函数预备知识

单位圆

三角函数的定义（直角三角形）

$$\sin\theta = \frac{\text{对边}}{斜边}, \quad \cos\theta = \frac{\text{邻边}}{斜边}, \quad \tan\theta = \frac{\text{对边}}{\text{邻边}} = \frac{\sin\theta}{\cos\theta}$$

🧠 记忆口诀："正弦对斜，余弦邻斜，正切对邻"。英文：SOH-CAH-TOA（Sine=Opposite/Hypotenuse 等）

特殊角三角函数值（必须背熟）

角度弧度$\sin$$\cos$$\tan$
$0°$$0$$0$$1$$0$
$30°$$\dfrac{\pi}{6}$$\dfrac{1}{2}$$\dfrac{\sqrt{3}}{2}$$\dfrac{\sqrt{3}}{3}$
$45°$$\dfrac{\pi}{4}$$\dfrac{\sqrt{2}}{2}$$\dfrac{\sqrt{2}}{2}$$1$
$60°$$\dfrac{\pi}{3}$$\dfrac{\sqrt{3}}{2}$$\dfrac{1}{2}$$\sqrt{3}$
$90°$$\dfrac{\pi}{2}$$1$$0$不存在
$180°$$\pi$$0$$-1$$0$

📌 记忆技巧：$\sin$ 值 $0° \to 90°$ 依次为 $0, \dfrac{1}{2}, \dfrac{\sqrt{2}}{2}, \dfrac{\sqrt{3}}{2}, 1$，即 $\dfrac{\sqrt{0}}{2}, \dfrac{\sqrt{1}}{2}, \dfrac{\sqrt{2}}{2}, \dfrac{\sqrt{3}}{2}, \dfrac{\sqrt{4}}{2}$。$\cos$ 反过来。

单位圆：三角函数的几何定义

基本三角恒等式

勾股恒等式（最重要）
$$\sin^2\theta + \cos^2\theta = 1$$
由此推出
$$\sin^2\theta = 1 - \cos^2\theta$$ $$1 + \tan^2\theta = \frac{1}{\cos^2\theta} \quad (\cos\theta \neq 0)$$

弧度与角度互换

$$\text{弧度} = \text{角度} \times \frac{\pi}{180°}, \quad \text{角度} = \text{弧度} \times \frac{180°}{\pi}$$

✅ 高中三角函数全部用弧度制。理解弧度：$\pi$ 弧度 $= 180°$，$2\pi$ 弧度 $= 360°$（一圈）。

例题 9.1 利用恒等式化简
已知 $\sin\theta = \dfrac{3}{5}$，且 $\theta$ 是第二象限角，求 $\cos\theta$ 和 $\tan\theta$。
1
由 $\sin^2\theta + \cos^2\theta = 1$：$\cos^2\theta = 1 - \left(\dfrac{3}{5}\right)^2 = 1 - \dfrac{9}{25} = \dfrac{16}{25}$
2
$\theta$ 在第二象限，$\cos\theta < 0$，故 $\cos\theta = -\dfrac{4}{5}$
3
$\tan\theta = \dfrac{\sin\theta}{\cos\theta} = \dfrac{3/5}{-4/5} = -\dfrac{3}{4}$

$\cos\theta = -\dfrac{4}{5}$，$\tan\theta = -\dfrac{3}{4}$

📌 各象限符号规律：第一象限三者均正；第二 $\sin$ 正；第三 $\tan$ 正；第四 $\cos$ 正。口诀：「一全正，二正弦，三正切，四余弦」

例题 9.2 特殊角计算
计算：$\sin 30° \cdot \cos 60° + \cos 30° \cdot \sin 60°$
1
代入特殊值：$\sin 30° = \dfrac{1}{2}$，$\cos 60° = \dfrac{1}{2}$，$\cos 30° = \dfrac{\sqrt{3}}{2}$，$\sin 60° = \dfrac{\sqrt{3}}{2}$
2
$= \dfrac{1}{2} \times \dfrac{1}{2} + \dfrac{\sqrt{3}}{2} \times \dfrac{\sqrt{3}}{2} = \dfrac{1}{4} + \dfrac{3}{4} = 1$

结果为 $1$

📌 该式其实正是高中的正弦和角公式：$\sin(A+B)=\sin A\cos B+\cos A\sin B$，即 $\sin(30°+60°)=\sin 90°=1$。你已经在预习高中内容了！
