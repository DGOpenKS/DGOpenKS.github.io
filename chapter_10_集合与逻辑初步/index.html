集合初步

高中第一章

集合的基本概念与符号

属于
$3 \in \{1,2,3\}$（$3$ 是集合的元素）
不属于
$4 \notin \{1,2,3\}$
子集
$A \subseteq B$（$A$ 的每个元素都在 $B$ 中）
真子集
$A \subsetneq B$（$A \subseteq B$ 且 $A \neq B$）
空集
$\emptyset$（不含任何元素，是任意集合的子集）
全集
$U$（研究范围内的所有元素之集合）
集合的两种表示法
$$A = \{1, 2, 3, 4, 5\} \quad \text{（列举法）}$$ $$B = \{x \mid x^2 \leq 4,\ x \in \mathbb{Z}\} = \{-2,-1,0,1,2\} \quad \text{（描述法）}$$

集合的三大运算

交集 $A \cap B$

同时属于 $A$ 和 $B$ 的元素组成的集合（公共部分）。

$\{1,2,3\}\cap\{2,3,4\}=\{2,3\}$

并集 $A \cup B$

属于 $A$ 或属于 $B$ 的所有元素（不重复计数）。

$\{1,2,3\}\cup\{2,3,4\}=\{1,2,3,4\}$

补集 $\complement_U A$

在全集 $U$ 中，不属于 $A$ 的所有元素。

若 $U=\{1,2,3,4,5\}$，$A=\{1,2\}$，则 $\complement_U A=\{3,4,5\}$

例题 10.1 集合运算（含不等式）
已知全集 $U = \mathbb{R}$，$A = \{x \mid -1 \leq x < 3\}$，$B = \{x \mid x < 1\}$，求 $A \cap B$，$A \cup B$，$\complement_U A$。
1
在数轴上标出：$A = [-1, 3)$，$B = (-\infty, 1)$
2
交集（公共区间）：两段区间的重叠部分为 $[-1, 1)$ $$A \cap B = \{x \mid -1 \leq x < 1\} = [-1, 1)$$
3
并集（合并覆盖）：将两区间合并（不重叠部分 $B$ 还包含 $(-\infty,-1)$） $$A \cup B = \{x \mid x < 3\} = (-\infty, 3)$$
4
补集（$A$ 之外的实数）：$A=[-1,3)$，其补集为两侧 $$\complement_U A = \{x \mid x < -1 \text{ 或 } x \geq 3\} = (-\infty, -1) \cup [3, +\infty)$$

$A \cap B = [-1, 1)$，$A \cup B = (-\infty, 3)$，$\complement_U A = (-\infty,-1)\cup[3,+\infty)$

例题 10.2 含参数的集合（高考常考）
已知 $A = \{1, 2, a\}$，$B = \{2, a^2\}$，且 $B \subseteq A$，求实数 $a$ 的值。
1
$B \subseteq A$ 意味着 $B$ 中每个元素都必须在 $A$ 中，已知 $2 \in A$ ✓，所以只需 $a^2 \in A$。
2
$A$ 中的元素为 $1, 2, a$，所以 $a^2$ 必须等于 $1$、$2$ 或 $a$ 之一。
3
情况一：$a^2 = 1$，则 $a = 1$ 或 $a = -1$。 若 $a=1$：$A=\{1,2,1\}$ 有重复元素，集合中元素互异，矛盾，舍去。 若 $a=-1$：$A=\{1,2,-1\}$，$B=\{2,1\}$，检查 $B \subseteq A$：$2\in A$ ✓，$1\in A$ ✓，成立 ✓
4
情况二：$a^2 = 2$，则 $a = \pm\sqrt{2}$。 若 $a=\sqrt{2}$：$A=\{1,2,\sqrt{2}\}$，$B=\{2,2\}$ 有重复，矛盾，舍去。 若 $a=-\sqrt{2}$：$A=\{1,2,-\sqrt{2}\}$，$B=\{2,2\}$ 有重复，矛盾，舍去。
5
情况三：$a^2 = a$，则 $a(a-1)=0$，$a=0$ 或 $a=1$。 若 $a=0$：$A=\{1,2,0\}$，$B=\{2,0\}$，$B\subseteq A$ ✓ 若 $a=1$：重复，已舍。

$a = -1$ 或 $a = 0$

📌 含参数集合题的关键：① 考虑所有情况；② 每种情况都要验证「元素互异」；③ 检查题目条件是否满足。
