# 公式与结构判断

公式判断的核心不是看表面像不像，而是判断对象是否真的匹配公式左边，以及当前应该展开还是分解。

## 平方相关公式

| 看到的对象 | 变形方向 | 方法 |
| --- | --- | --- |
| `(a+b)^2` | 展开 | 拆成 `a^2 + 2ab + b^2` |
| `(a-b)^2` | 展开 | 拆成 `a^2 - 2ab + b^2` |
| `a^2-b^2` | 分解 | 写成 `(a-b)(a+b)` |

判断重点：

```text
整体有没有被平方？
是不是两个平方相减？
中间项是不是正好等于 ±2ab？
```

## 立方相关公式

| 看到的对象 | 变形方向 | 方法 |
| --- | --- | --- |
| `(a+b)^3` | 展开 | 三个相同因式相乘 |
| `(a-b)^3` | 展开 | 把 `a-b` 看成 `a+(-b)` |
| `a^3+b^3` | 分解 | 先找一次因子 `a+b`，再让混合项抵消 |
| `a^3-b^3` | 分解 | 先找一次因子 `a-b`，再让混合项抵消 |

判断重点：

```text
它是一个整体被立方，还是两个完整立方相加减？
如果是整体立方，就展开。
如果是两个完整立方相加减，就分解。
```

## 使用条件

| 方法 | 必须满足 |
| --- | --- |
| 完全平方 | 首尾是平方，中间项匹配 `±2ab` |
| 平方差 | 两项相减，且都能写成平方 |
| 和的立方/差的立方 | 整个括号被三次方 |
| 立方和/立方差 | 两端都是完整三次方 |

## 常见误用

| 错误 | 错因标签 | 检查方式 |
| --- | --- | --- |
| `(a+b)^2=a^2+b^2` | 漏交叉项 | 写成 `(a+b)(a+b)` |
| `(a-b)^2=a^2-b^2` | 对象混淆 | 判断是差的平方还是平方的差 |
| `a^2+b^2=(a+b)(a-b)` | 符号条件错误 | 平方差必须是相减 |
| `(a+b)^3=a^3+b^3` | 整体立方误解 | 写成三个括号相乘 |
| `a^3+b^3=(a+b)(a^2+ab+b^2)` | 混合项未抵消 | 乘回去检查 |

## 检查方式

公式使用后，要能回到原式：

```text
展开公式：重新合并同类项，看是否得到展开结果。
分解公式：把因式乘回去，看是否回到原式。
```

尤其是立方和、立方差，二次因子中间项的符号不是凭记忆决定，而是由混合项能否抵消决定。
