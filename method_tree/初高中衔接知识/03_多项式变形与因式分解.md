# 多项式变形与因式分解

多项式变形的核心原则是：

```text
值不变，结构变清楚
```

因式分解的核心目标是：

```text
加法结构 → 乘法结构
```

## 方法选择顺序

面对一个多项式，先按这个顺序判断：

```text
每一项都有共同部分吗？
→ 有没有直接公式结构？
→ 是不是二次三项式？
→ 有没有重复整体？
→ 四项或更多能不能分组？
→ 是否需要配方制造平方结构？
→ 分解完能不能乘回原式？
```

## 结构信号与方法

| 结构信号 | 优先方法 | 检查重点 |
| --- | --- | --- |
| 每一项都有共同部分 | 提公因式 | 公因式是否提完整，括号是否还能继续分解 |
| 两项相减且都是平方 | 平方差 | 是否真能写成 `A^2-B^2` |
| 三项像平方展开 | 完全平方 | 中间项是否是 `±2AB` |
| 两个完整立方相加减 | 立方和/立方差 | 一次因子和二次因子混合项是否抵消 |
| 二次三项式 | 十字相乘或配方 | 乘积和一次项是否同时匹配 |
| 重复整体 | 整体代换 | 换元后是否换回原整体 |
| 四项或更多 | 分组分解 | 分组后是否制造出共同括号 |
| 需要看顶点或平方结构 | 配方 | 加了多少就要减回多少 |

## 常用方法节点

### 提公因式

适用：每一项都有共同因子。

检查：

```text
提出来后，乘回去是否等于原式？
括号里是否还能继续分解？
```

### 公式法

适用：式子完整匹配平方差、完全平方、立方和、立方差。

检查：

```text
是否只看表面相似？
公式左边结构是否真的满足？
```

### 十字相乘

适用：二次三项式。

核心检查：

```text
二次项乘积对不对？
常数项乘积对不对？
交叉乘积合成的一次项对不对？
```

首项系数为 `1` 时，可以先找：

```text
m+n=p
mn=q
```

首项系数不是 `1` 时，必须同时拆二次项和常数项，再看交叉项。

### 分组分解

适用：四项或更多项。

关键不是随便分组，而是分组后出现同一个括号。

### 整体代换

适用：同一个整体反复出现。

例如：

```text
(x+1)^2 - 5(x+1) + 6
```

先把 `x+1` 看成一个整体，避免一开始展开散掉结构。

### 配方

适用：需要制造完整平方结构。

检查：

```text
补了多少，就要减回多少。
配方后是否出现平方差或顶点结构？
```

## 最后检查

因式分解不是写完就结束。必须能乘回原式。

```text
分解结果 → 展开 → 合并同类项 → 原式
```

如果乘回去一次项、常数项或符号不一致，说明方法选择或计算有错。
