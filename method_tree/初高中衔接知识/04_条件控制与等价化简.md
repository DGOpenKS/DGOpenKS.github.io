# 条件控制与等价化简

进入分式和根式以后，代数对象不只是表达式形式，还包括成立条件。

```text
代数对象 = 表达式形式 + 成立条件
等价化简 = 形式相等 + 条件一致
```

## 条件检查顺序

化简前先问：

```text
有没有分母？
有没有根号？
有没有约分？
有没有多个条件要同时满足？
```

标准流程：

```text
先写原式条件
→ 再做代数变形
→ 标出可能改变条件的步骤
→ 检查条件有没有丢失或扩大
→ 最后把结果和条件一起写出
```

## 分式条件

分式的基本条件：

```text
分母不能为 0
```

约分时要记住：

```text
约掉的是共同因子，不是原式条件。
```

如果原分母中某个因子被约掉，它造成的禁点仍然必须保留。

## 根式条件

实数范围内：

```text
被开方数 ≥ 0
平方根结果 ≥ 0
```

所以：

```text
sqrt(A^2) = |A|
```

只有已经知道 `A ≥ 0` 时，才可以继续写成 `A`；如果 `A < 0`，应写成 `-A`。

## 复合条件

一个表达式可能同时有根号和分母。每个限制都要写出来，最后取共同范围。

检查顺序：

```text
先看每一个根号
→ 写出被开方数 ≥ 0
→ 再看每一个分母
→ 写出分母 ≠ 0
→ 如果有约分，保留原分母条件
→ 最后取所有条件的共同部分
```

## 常见风险

| 风险 | 错因标签 | 正确动作 |
| --- | --- | --- |
| 约掉分母因子后忘记禁点 | 条件丢失 | 先写原分母不为 `0` |
| 把 `sqrt(A^2)` 写成 `A` | 机械开方 | 先写成 `|A|` |
| 只保留一个条件 | 复合条件遗漏 | 根号、分母、约分条件全部合并 |
| 化简式有值就以为原式有值 | 原式条件扩大 | 代回原表达式判断 |

## 输出格式

等价化简的最终结果应包含两部分：

```text
化简后的形式
原式允许的条件
```

例如：

```text
原式 = 化简式，且 x ≠ ...
```
