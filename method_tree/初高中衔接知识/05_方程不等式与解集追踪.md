# 方程不等式与解集追踪

方程和不等式的结果不是简单的计算产物，而是解集。

```text
方程的解：让左右相等的输入值
不等式的解：让关系成立的范围
约束：从一开始就不能进入讨论的值
```

## 解集追踪流程

```text
允许范围
→ 变形过程
→ 候选结果
→ 条件筛选
→ 代回原式
→ 最终解集
```

## 方程变形

通常安全：

```text
两边同时加减同一个式子
两边同时乘除确定非零的数
```

需要检查：

```text
两边平方
乘以含未知数的式子
约掉可能为 0 的因式
根式方程平方
分式方程去分母
```

这些操作可能产生增根，也可能丢失解。

## 分式根式方程

标准流程：

```text
写原式约束
→ 变形求候选解
→ 筛掉不满足约束的候选解
→ 代回原式确认
```

根式方程要注意：

```text
根号内 ≥ 0
平方根所在一边必须非负
平方后可能产生增根
```

分式方程要注意：

```text
原分母不能为 0
去分母后候选解不能落在禁点上
```

## 不等式变形

不等式中乘除法最容易出错。

```text
乘除正数：方向不变
乘除负数：方向改变
乘除符号不确定的式子：不能直接操作
```

如果乘除对象含未知数，通常要改用分情况讨论或移到一边后做符号分析。

## 二次不等式

流程：

```text
移到一边，与 0 比较
→ 能分解则先分解
→ 找零点
→ 用零点分区间
→ 判断每段符号
→ 根据 >, ≥, <, ≤ 决定端点是否保留
```

零点不是答案本身，而是数轴分段的边界。

## 分式不等式

流程：

```text
移到一边，与 0 比较
→ 通分成一个分式
→ 找分子零点
→ 找分母禁点
→ 零点和禁点一起分区间
→ 判断每段符号
→ 分子零点看题目是否取，分母禁点永远不取
```

## 最终检查

写答案前问：

```text
候选解是否满足原式条件？
变形中是否平方、去分母、约掉含未知数因子？
不等式端点是否能取？
分母禁点是否被排除？
答案是点、区间，还是几个区间的并集？
```
