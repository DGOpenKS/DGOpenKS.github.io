# 初高中衔接方法树

这棵方法树不是技巧清单，而是帮助你面对一个数学对象时知道怎样下手。

初高中衔接部分的核心变化是：不要只问“怎么算”，还要问：

```text
这是什么对象？
它露出了什么结构信号？
当前目标是什么？
哪种方法能让结构更清楚？
变形后条件或解集有没有改变？
最后能不能检查回去？
```

## 总判断链

```text
对象识别
→ 结构判断
→ 方法选择
→ 条件控制
→ 解集追踪
→ 回代或乘回检查
```

这条链对应初高中衔接知识树：

| 方法树节点 | 对应知识内容 | 解决的问题 |
| --- | --- | --- |
| 对象识别 | Chapter 01 | 同一个式子可能是方程、函数、表达式或关系 |
| 公式与结构判断 | Chapter 02-1, Chapter 02-2 | 先判断对象，再决定展开或分解 |
| 多项式变形与因式分解 | Chapter 03 | 根据结构信号选择变形方法 |
| 条件控制与等价化简 | Chapter 04 | 化简时形式和条件都要保留 |
| 方程不等式与解集追踪 | Chapter 05 | 计算结果只是候选解，最终要筛选 |

## 方法树使用方式

遇到一道题时，不要先背方法名称。按顺序问：

1. 它现在是表达式、方程、不等式，还是函数对象？
2. 题目目标是展开、分解、化简、求解、判断正负，还是研究零点？
3. 式子里有没有公因式、平方结构、立方结构、重复整体、分母或根号？
4. 这一步变形是否可能改变条件或解集？
5. 最后能不能乘回、展开回去，或代回原式检查？

## 最重要的原则

方法树的目标不是让你记住更多名字，而是让你形成稳定的判断顺序：

```text
先看对象，再看结构。
先判断能不能用，再决定怎么做。
做完以后，一定检查条件和结果。
```
