# 对象识别练习

对象识别练习训练第一反应：先判断它是什么，再决定怎么做。

## 练习目标

学习者应能判断：

```text
表达式
方程
不等式
函数对象
带条件表达式
含参数对象族
```

## 基础练习

给出一个对象，选择它当前的身份。

示例：

```text
x^2 - 4x + 3
x^2 - 4x + 3 = 0
y = x^2 - 4x + 3
x^2 - 4x + 3 > 0
(x^2 - 1)/(x - 1)
sqrt((x-2)^2)
```

反馈重点：

```text
同一个式子可以有不同身份。
身份不同，目标和方法也不同。
```

## 变式练习

同一个式子放入不同任务中，让学习者判断该优先看什么。

| 任务 | 应先判断 |
| --- | --- |
| 求 `x^2-4x+3=0` 的解 | 方程的解 |
| 判断 `x^2-4x+3>0` | 不等式的符号范围 |
| 研究 `y=x^2-4x+3` | 函数对象的零点、正负、顶点 |
| 化简 `(x^2-1)/(x-1)` | 原分母条件 |

## 错误诊断

| 错误表现 | 错因标签 | 反馈重点 |
| --- | --- | --- |
| 看到二次式就直接因式分解，不看题目目标 | `task-mismatch` | 方法要服务当前任务 |
| 把化简后的表达式当成原式 | `range-expanded` | 表达式形式和条件一起构成对象 |
| 把方程的解当成函数全部信息 | `function-expression-mix` | 解只是零点，函数还可研究正负、顶点、定义域 |

## 过关标准

学习者能用一句话说明：

```text
这是一个什么对象？
当前目标是什么？
我为什么先看这个结构？
```
