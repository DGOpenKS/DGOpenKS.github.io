# 结构判断练习

结构判断练习训练的是：看到式子时，先看它露出了什么方法信号。

## 练习目标

学习者应能识别：

```text
公因式
完全平方
平方差
整体立方
立方和差
二次三项式
重复整体
可分组结构
分母禁点
根式非负
```

## 基础识别

给出式子，只判断结构，不计算。

示例：

```text
6x^2 - 9x
x^2 - 25
x^2 + 10x + 25
x^2 + 8x + 15
(x+1)^2 - 5(x+1) + 6
xy + 3x + 2y + 6
(a+b)^3
a^3 + b^3
sqrt((x-2)^2)
(x^2-4)/(x^2-3x+2)
```

反馈重点：

```text
不要先算。
先说它像什么结构，以及为什么。
```

## 对比练习

把容易混淆的对象放在一起。

| A | B | 要分清 |
| --- | --- | --- |
| `(a-b)^2` | `a^2-b^2` | 差的平方 vs 平方的差 |
| `(a+b)^3` | `a^3+b^3` | 整体立方 vs 立方和 |
| `x^2+10x+25` | `x^2+8x+25` | 中间项是否匹配 |
| `(x+1)/(x+2)` | `(x-2)(x+2)/((x-1)(x-2))` | 项不能约，因子可以约 |

## 变式练习

要求学习者写出结构判断句：

```text
这个式子每一项都有 ______，所以先 ______。
这个式子是两个平方 ______，所以可以 ______。
这个式子里重复出现 ______，所以先把它看成一个整体。
这个式子有分母，所以先写 ______。
```

## 错误诊断

| 错误表现 | 错因标签 | 反馈重点 |
| --- | --- | --- |
| 首尾是平方就判完全平方 | `middle-term-mismatch` | 还要检查中间项 |
| 看到两个平方就用平方差 | `difference-required` | 平方差必须相减 |
| 看到复杂式子就展开 | `whole-expanded-too-early` | 重复整体要先保留 |
| 看到相同字母就约掉 | `factor-term-cancel` | 只能约共同因子 |

## 过关标准

学习者不计算也能说出：

```text
它有什么结构信号？
这个信号对应什么方法？
这个方法有没有使用条件？
```
