# 方法执行练习

方法执行练习训练在已经判断出结构后，能把方法做完整、做准确。

## 练习目标

学习者应能执行：

```text
公式展开
公式分解
提公因式
十字相乘
分组分解
整体代换
配方
分式化简
根式化简
零点分区间
```

## 公式执行

练习族：

```text
(x+n)^2
(x-n)^2
x^2-n^2
(a+b)^3
(a-b)^3
x^3+n^3
x^3-n^3
```

反馈重点：

```text
展开看交叉项是否完整。
分解看乘回去混合项是否抵消。
```

## 因式分解执行

练习族：

```text
kx^2 + kpx
x^2 + px + q
ax^2 + bx + c
A^2 - B^2
A^2 ± 2AB + B^2
xy + ay + bx + ab
(x+n)^2 + p(x+n) + q
```

反馈重点：

```text
分解结果必须能乘回原式。
十字相乘必须同时检查二次项、常数项和一次项。
```

## 化简执行

练习族：

```text
(可分解分子)/(可分解分母)
sqrt((一次式)^2)
含根号和分母的复合表达式
```

反馈重点：

```text
先写原式条件，再做化简。
约分只约因子，条件不能丢。
```

## 不等式执行

练习族：

```text
(x-a)(x-b) > 0
(x-a)(x-b) >= 0
(x-a)/(x-b) > 0
(x-a)/(x-b) >= 0
```

反馈重点：

```text
零点和禁点都要放到数轴上。
严格不等号和非严格不等号决定端点取舍。
分母禁点永远不取。
```

## 错误诊断

| 错误表现 | 错因标签 | 反馈重点 |
| --- | --- | --- |
| 十字相乘只看常数项 | `constant-only-factor` | 还要检查一次项 |
| 配方后值改变 | `complete-square-balance` | 补了多少就减回多少 |
| 分组后没有共同括号 | `grouping-no-common-bracket` | 分组目标是制造共同结构 |
| 公式分解后不乘回检查 | `no-multiply-back-check` | 乘回去是最终验证 |

## 过关标准

学习者能完成步骤，并能解释：

```text
我为什么选这个方法？
这个方法执行时最容易错在哪里？
我如何检查结果？
```
