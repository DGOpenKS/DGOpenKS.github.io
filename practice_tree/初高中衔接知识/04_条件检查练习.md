# 条件检查练习

条件检查练习专门训练：计算前后，哪些值允许留下。

## 练习目标

学习者应能处理：

```text
分母不为 0
根式被开方数非负
平方根结果非负
约分后禁点保留
根式方程增根筛选
分式方程禁点筛选
不等式端点取舍
分式不等式分母禁点
```

## 原式条件练习

只要求写允许范围，不化简。

示例：

```text
(x^2-1)/(x-1)
sqrt(x+2)/(x^2-1)
sqrt((x-2)^2)
1/(x-1) > 2
```

反馈重点：

```text
条件来自原式，不是化简后补出来的。
```

## 化简后条件练习

要求先化简，再写完整条件。

示例：

```text
(x^2-4)/(x^2-3x+2)
(x^2-9)/(x^2-x-6)
sqrt((x-a)^2)
```

反馈重点：

```text
化简式可能看不出被约掉的禁点。
sqrt(A^2) 先写成 |A|。
```

## 方程筛选练习

要求写：

```text
原式约束
候选解
筛选过程
最终解
```

适合题型：

```text
根式方程
分式方程
含平方变形的方程
```

反馈重点：

```text
候选解不是最终答案。
必须代回原式或用原式条件筛选。
```

## 不等式端点练习

要求判断端点是否保留。

适合题型：

```text
(x-a)(x-b) > 0
(x-a)(x-b) >= 0
(x-a)/(x-b) > 0
(x-a)/(x-b) >= 0
```

反馈重点：

```text
分子零点能否取，取决于不等号是否允许等于 0。
分母禁点永远不能取。
```

## 错误诊断

| 错误表现 | 错因标签 | 反馈重点 |
| --- | --- | --- |
| 约分后忘记被约掉的禁点 | `domain-lost` | 先写原分母条件 |
| 把 `sqrt(A^2)` 直接写成 `A` | `sqrt-absolute-missed` | 开方结果必须非负 |
| 根式方程平方后不验根 | `extraneous-root` | 平方可能带来增根 |
| 分式不等式取了分母零点 | `forbidden-point-included` | 分母为 0 时原式无意义 |

## 过关标准

学习者能主动写出：

```text
原式允许范围
变形后的候选结果
最终保留或排除的理由
```
