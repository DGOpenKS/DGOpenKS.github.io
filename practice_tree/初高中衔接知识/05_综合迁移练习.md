# 综合迁移练习

综合迁移练习训练把前面的结构判断、方法执行和条件检查放到同一道题里。

## 练习目标

学习者应能把结构方法用于：

```text
解方程
看函数零点
判断函数正负
解二次不等式
解分式不等式
化简分式
判断表达式是否等价
```

## 迁移路径

| 知识能力 | 迁移到 |
| --- | --- |
| 因式分解 | 方程求解、函数零点、不等式分区间、分式化简 |
| 完全平方 | 配方、顶点、最值、平方结构判断 |
| 平方差 | 因式分解、约分、二次不等式 |
| 立方和差 | 三次式分解、降次 |
| 条件控制 | 分式根式方程、等价化简、分式不等式 |
| 解集追踪 | 候选解筛选、端点取舍、禁点排除 |

## 综合题型

### 从表达式到方程

示例结构：

```text
x^2 + px + q = 0
```

训练：

```text
先分解
再用乘积为 0
最后写解
```

### 从表达式到函数零点

示例结构：

```text
f(x)=x^2+px+q
```

训练：

```text
分解后看哪些 x 让 f(x)=0
```

### 从表达式到不等式

示例结构：

```text
x^2 + px + q > 0
```

训练：

```text
分解
找零点
分区间
判断符号
```

### 从因式分解到分式化简

示例结构：

```text
(可分解二次式)/(含公共因子的式子)
```

训练：

```text
先写原分母条件
再分解约分
最后保留禁点
```

### 从分式到分式不等式

示例结构：

```text
(x-a)/(x-b) >= 0
```

训练：

```text
分子零点
分母禁点
数轴分区间
端点取舍
```

## 反馈重点

综合练习反馈不能只盯计算错误，要指出是哪一层出了问题：

```text
对象识别错
结构判断错
方法执行错
条件检查漏
最终解集写法错
```

## 过关标准

学习者能完整说明：

```text
我先判断了什么对象。
我看到了什么结构信号。
我选择了什么方法。
我保留了哪些条件。
我怎样检查最终结果。
```
