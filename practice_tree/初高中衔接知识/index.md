# 初高中衔接练习树

这棵练习树不是题目堆叠，而是按照能力递进组织练习。

核心路径是：

```text
识别对象
→ 判断结构
→ 执行方法
→ 检查条件
→ 综合迁移
```

每一类练习都要让学习者知道自己在练什么，而不是只追求算出答案。

## 练习层级

| 层级 | 训练目标 | 典型问题 |
| --- | --- | --- |
| 识别对象 | 看清当前数学对象 | 这是表达式、方程、不等式，还是函数对象？ |
| 判断结构 | 看出方法信号 | 有没有公因式、平方差、完全平方、重复整体、分母、根号？ |
| 执行方法 | 正确完成变形 | 展开、分解、约分、配方、通分、分区间 |
| 检查条件 | 保留原式限制 | 禁点、非负性、增根、端点是否处理正确？ |
| 综合迁移 | 把结构用于任务 | 解方程、看零点、解不等式、化简分式、判断正负 |

## 练习设计原则

1. 每题先服务一个明确能力，不为了难而难。
2. 先练判断，再练计算。
3. 固定例题用于揭示结构，参数化练习用于巩固熟练度。
4. 反馈要指出错因结构，而不是只给答案。
5. 综合题必须保留条件追踪，不把候选结果直接当答案。
