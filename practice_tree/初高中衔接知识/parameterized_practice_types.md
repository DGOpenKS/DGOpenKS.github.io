# 参数化练习类型

参数化练习用于生成同一结构下的多组题。它适合巩固熟练度，但不能替代固定例题对结构来源的解释。

## 公式类

### 完全平方

模板：

```text
(x+n)^2
(x-n)^2
(mx+n)^2
(mx-n)^2
```

参数建议：

```text
m,n 取小正整数
避免系数过大导致计算遮住结构
```

反馈重点：

```text
中间项来自两次交叉乘积。
```

### 平方差

模板：

```text
x^2-n^2
m^2x^2-n^2
(x+a)^2-b^2
```

反馈重点：

```text
必须是两个平方相减。
```

### 立方展开

模板：

```text
(a+b)^3
(a-b)^3
(x+n)^3
(x-n)^3
```

反馈重点：

```text
三个相同因式相乘，系数来自 1,3,3,1。
```

### 立方和差

模板：

```text
x^3+n^3
x^3-n^3
(mx)^3-n^3
```

参数建议：

```text
n 选可直接看成立方的数，如 1, 8, 27, 64, 125
```

反馈重点：

```text
先识别完整立方，再找一次因子。
```

## 因式分解类

### 提公因式

模板：

```text
kx^2+kpx
kx^2+kpx+kq
```

反馈重点：

```text
先提共同部分，括号里还能分解就继续。
```

### 二次三项式

模板：

```text
x^2+(m+n)x+mn
x^2+(m-n)x-mn
(ax+b)(cx+d) 展开后反向给题
```

反馈重点：

```text
同时检查积和和，首项系数不是 1 时还要检查交叉乘积。
```

### 分组分解

模板：

```text
ax+ay+bx+by
xy+ay+bx+ab
```

反馈重点：

```text
分组后要制造共同括号。
```

### 整体代换

模板：

```text
(x+a)^2+p(x+a)+q
(x^2+x)^2+p(x^2+x)+q
```

反馈重点：

```text
重复整体先保留，不要急着展开。
```

## 条件控制类

### 分式化简

模板：

```text
((x-a)(x+b))/((x-a)(x+c))
```

反馈重点：

```text
被约掉的因子仍然给出原式禁点。
```

### 根式化简

模板：

```text
sqrt((x-a)^2)
sqrt((kx+b)^2)
```

反馈重点：

```text
先写绝对值，再按符号条件分情况。
```

### 复合条件

模板：

```text
sqrt(x+a)/(x-b)
sqrt(x+a)/((x-b)(x-c))
```

反馈重点：

```text
根号条件和分母条件要同时满足。
```

## 解集追踪类

### 二次不等式

模板：

```text
(x-a)(x-b) > 0
(x-a)(x-b) >= 0
(x-a)(x-b) < 0
(x-a)(x-b) <= 0
```

反馈重点：

```text
零点分区间，端点是否取由不等号决定。
```

### 分式不等式

模板：

```text
(x-a)/(x-b) > 0
(x-a)/(x-b) >= 0
(x-a)/(x-b) < 0
(x-a)/(x-b) <= 0
```

反馈重点：

```text
分子零点可以按题目判断是否取，分母禁点永远不取。
```

## 生成限制

参数化生成时要避免：

```text
参数导致结构退化
数字过大遮住结构
分母因子重复造成额外复杂性
根式条件和分母条件无交集却没有说明
不等式端点重合导致超出当前训练目标
```
