高中数学与初中数学的区别

思维跨越

思维方式的根本转变

很多同学进入高中后成绩下降，并不是因为不够努力，而是还在用初中的思维方式学高中数学。理解这个差异，是高中学好数学的第一步。

对比维度初中数学高中数学
研究对象具体数字和图形变量之间的规律与关系
核心工具方程（求某个值）函数（描述整个变化过程）
思维层次算术思维、几何直觉抽象思维、符号推理
典型问题$x^2=4$ 解为多少？$f(x)=x^2$ 有哪些性质？
图形作用辅助计算与代数等价，数形结合
难点来源计算繁琐概念抽象、逻辑严密

初中思维：「解题得到答案」

解方程 $x^2 - 4 = 0$

初中做法：移项→开方，得 $x = \pm 2$，问题结束。

💡 初中关注的是「这道题的答案是多少」，目标是一个具体数字。

高中思维：「研究整个规律」

研究 $f(x) = x^2 - 4$

高中要问：定义域？值域？单调区间？图像形状？零点位置？最小值？奇偶性？

✅ 高中关注的是「这个函数的全貌」，同一个式子能挖掘出几十个问题。

高中数学的三大核心思想

① 函数思想

用 $y=f(x)$ 描述两个量的对应关系，研究变化趋势。高中80%的内容本质上都是在研究函数。

② 数形结合

代数问题转化为几何图形来直观理解，几何结论用代数式精确表达。两种语言互相翻译。

③ 分类讨论

当出现字母参数、绝对值、根号等情况时，需要针对不同情形分别讨论，不能一刀切。

例题 1.1 感受初高中思维差异
【初中问题】已知一个矩形的长比宽多 $3$，面积为 $28$，求长和宽。
【高中问题】若矩形的宽为 $x$，面积为 $f(x) = x(x+3)$，求使面积超过 $28$ 的宽的范围。
1
初中解法：设宽为 $x$，则长为 $x+3$，面积方程：$x(x+3)=28$，即 $x^2+3x-28=0$，解得 $x=4$（舍负），长 $=7$。问题结束。
2
高中分析：要研究 $f(x)=x(x+3)>28$，即 $x^2+3x-28>0$，即 $(x-4)(x+7)>0$，因为宽 $x>0$，所以解集为 $x>4$。

高中问题的答案是一个「范围」，而不是一个数。这正是函数思想——关注变量变化的整个区间。

实数与运算基础

数的世界

实数的分类体系

⚠️ 常见误区：$0$ 是整数、也是自然数、也是有理数，但不是正整数，也不是负整数。$0$ 既不是正数也不是负数。

绝对值的本质

绝对值描述的是数轴上某点到原点的距离，距离永远非负。

绝对值定义
$$|a| = \begin{cases} a & \text{当 } a \geq 0 \\ -a & \text{当 } a < 0 \end{cases} \qquad \text{几何意义：} |a-b| \text{ 表示数轴上 } a \text{ 与 } b \text{ 的距离}$$
非负性
$|a| \geq 0$，且 $|a|=0 \Leftrightarrow a=0$
对称性
$|-a|=|a|$
绝对值与平方根
$\sqrt{a^2}=|a|$（非常重要！）
三角不等式
$|a+b| \leq |a|+|b|$
乘法
$|ab|=|a||b|$
距离
$|a-b|$ = $a$、$b$ 之间的距离
例题 2.1 绝对值方程
解方程 $|2x - 3| = 5$
1
绝对值方程 $|A| = c$（$c \geq 0$）等价于 $A = c$ 或 $A = -c$，分两种情况讨论：
2
情况一：$2x - 3 = 5$，解得 $2x = 8$，$x = 4$
3
情况二：$2x - 3 = -5$，解得 $2x = -2$，$x = -1$

解为 $x = 4$ 或 $x = -1$。

🔍 几何理解：$|2x-3|=5$ 等价于 $2x$ 与 $3$ 的距离为 $5$，即 $2x=8$ 或 $2x=-2$。

例题 2.2 根式化简（含绝对值）
化简 $\sqrt{a^2 - 2a + 1}$，当 $a < 1$ 时。
1
首先识别被开方数的结构：$a^2 - 2a + 1 = (a-1)^2$（完全平方式）
2
因此 $\sqrt{a^2-2a+1} = \sqrt{(a-1)^2} = |a-1|$
3
当 $a < 1$ 时，$a - 1 < 0$，所以 $|a-1| = -(a-1) = 1 - a$

当 $a < 1$ 时，$\sqrt{a^2-2a+1} = 1-a$。

⚠️ 最常见错误：直接写 $\sqrt{(a-1)^2}=a-1$，忽视了 $a<1$ 导致 $a-1<0$，结果应取绝对值。

例题 2.3 指数运算综合
计算 $\dfrac{(2^3 \times 2^{-1})^2}{2^4 \div 2^2}$
1
处理分子：括号内 $2^3 \times 2^{-1} = 2^{3+(-1)} = 2^2$，再平方 $(2^2)^2 = 2^4$
2
处理分母：$2^4 \div 2^2 = 2^{4-2} = 2^2 = 4$
3
整体：$\dfrac{2^4}{2^2} = 2^{4-2} = 2^2 = 4$

结果为 $4$。

指数六大法则（高中指数函数的基础）

同底相乘（指数相加）
$a^m \cdot a^n = a^{m+n}$
同底相除（指数相减）
$\dfrac{a^m}{a^n} = a^{m-n}$
幂的乘方（指数相乘）
$(a^m)^n = a^{mn}$
积的乘方
$(ab)^n = a^n b^n$
零次幂
$a^0 = 1 \quad (a \neq 0)$
负整数次幂
$a^{-n} = \dfrac{1}{a^n} \quad (a \neq 0)$

✅ 高中还会学到：$a^{1/n} = \sqrt[n]{a}$（分数次幂），这六条法则同样适用，理解这里的规律对高中指数函数至关重要。

整式与因式分解

代数基础

必须烂熟于心的公式

完全平方（和）
$(a+b)^2 = a^2 + 2ab + b^2$
完全平方（差）
$(a-b)^2 = a^2 - 2ab + b^2$
平方差
$(a+b)(a-b) = a^2 - b^2$
立方和
$a^3+b^3=(a+b)(a^2-ab+b^2)$
立方差
$a^3-b^3=(a-b)(a^2+ab+b^2)$
三项展开
$(a+b+c)^2=a^2+b^2+c^2+2ab+2bc+2ca$

因式分解的三种主要方法

① 提公因式法

找出各项的公共因子，整体提出。
例：$6x^2+9x = 3x(2x+3)$

② 公式法

利用完全平方、平方差等公式。
例：$x^2-9=(x+3)(x-3)$

③ 十字相乘法

适用于 $ax^2+bx+c$，找两数之积为 $ac$、和为 $b$。

🔗 为什么因式分解这么重要？高中中，解二次方程靠因式分解；化简三角函数表达式靠因式分解；求函数零点靠因式分解；导数运算中提取公因子也靠因式分解。

例题 3.1 十字相乘法详解
分解因式：$2x^2 + 7x + 3$
1
对于 $ax^2+bx+c$，需要找两个数 $p, q$，使得 $p \cdot q = a \cdot c = 2 \times 3 = 6$，且 $p + q = b = 7$。
2
列举积为 $6$ 的正整数对：$(1,6),(2,3)$。检验和：$1+6=7$ ✓（找到了）
3
拆中间项：$2x^2 + 7x + 3 = 2x^2 + x + 6x + 3$
4
分组提取公因子：$= x(2x+1) + 3(2x+1) = (2x+1)(x+3)$

$2x^2+7x+3 = (2x+1)(x+3)$

🔍 验证：$(2x+1)(x+3) = 2x^2+6x+x+3 = 2x^2+7x+3$ ✓

例题 3.2 综合运用（换元法）
分解因式：$(x^2+x)^2 - 8(x^2+x) + 12$
1
观察到 $x^2+x$ 反复出现，令 $t = x^2+x$，原式变为：$t^2 - 8t + 12$
2
对 $t^2-8t+12$ 分解：找积为 $12$、和为 $-8$ 的两数：$(-2) \times (-6) = 12$，$(-2)+(-6)=-8$ ✓
3
$t^2-8t+12 = (t-2)(t-6)$
4
回代 $t = x^2+x$：$(x^2+x-2)(x^2+x-6)$
5
继续分解：$x^2+x-2=(x+2)(x-1)$，$x^2+x-6=(x+3)(x-2)$

$(x+2)(x-1)(x+3)(x-2)$

📌 换元法是高中数学的重要解题策略，在三角函数、数列等章节中频繁使用。

例题 3.3 完全平方式的配凑
判断 $x^2 + 6x + 10$ 能否分解为整系数因式之积，并配成完全平方形式。
1
尝试十字相乘：找积为 $10$、和为 $6$ 的两整数，$(2,5)$：$2+5=7≠6$，$(1,10)$：$1+10=11≠6$，均不满足。
2
结论：不能在整数范围内分解。用配方法改写： $$x^2+6x+10 = (x^2+6x+9)+1 = (x+3)^2+1$$
3
由于 $(x+3)^2 \geq 0$，所以 $(x+3)^2+1 \geq 1 > 0$，说明该二次式对所有实数 $x$ 恒大于零，即方程 $x^2+6x+10=0$ 无实数解（判别式 $\Delta=36-40=-4 < 0$）。

$x^2+6x+10=(x+3)^2+1$，在实数范围内不可分解。最小值为 $1$（当 $x=-3$ 时取得）。

📌 配方法是高中求二次函数最值的核心方法，也是顶点坐标公式的来源。

分式与根式

有理化

分式的核心原则

分式 $\dfrac{A}{B}$ 有意义的前提：$B \neq 0$。

分式基本性质
$$\frac{A}{B} = \frac{A \cdot C}{B \cdot C} \quad (C \neq 0)$$
运算法则
$$\frac{a}{b} \pm \frac{c}{d} = \frac{ad \pm bc}{bd}, \quad \frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}$$

⚠️ 分母不能为零，这在高中求函数定义域时是第一个要检查的条件。

根式运算与有理化

根式基本运算
$$\sqrt{a} \cdot \sqrt{b} = \sqrt{ab}, \quad \frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}} \quad (a,b \geq 0)$$
有理化分母
$$\frac{1}{\sqrt{a}} = \frac{\sqrt{a}}{a}, \quad \frac{1}{\sqrt{a}-\sqrt{b}} = \frac{\sqrt{a}+\sqrt{b}}{a-b}$$

📌 有理化的目的：将分母中的根号消去，使表达式更简洁，便于计算。

例题 4.1 化简分式（先因式分解）
化简：$\dfrac{x^2 - 4}{x^2 - x - 2}$，并指出其无意义的条件。
1
分解分子：$x^2-4 = (x+2)(x-2)$（平方差公式）
2
分解分母：$x^2-x-2$，找积为 $-2$、和为 $-1$ 的两数：$-2$ 和 $1$，故 $x^2-x-2=(x-2)(x+1)$
3
约分：$\dfrac{(x+2)(x-2)}{(x-2)(x+1)} = \dfrac{x+2}{x+1}$（其中 $x \neq 2$）
4
分母无意义的条件：分母 $x^2-x-2=0$ 时，即 $x=2$ 或 $x=-1$ 时，分式无意义。

化简结果为 $\dfrac{x+2}{x+1}$，其中 $x \neq 2$ 且 $x \neq -1$。

⚠️ 约分后 $x \neq 2$ 的限制不能丢失，即使结果中分母不含 $(x-2)$，也必须注明。

例题 4.2 分母有理化
化简：$\dfrac{3}{\sqrt{5} - \sqrt{2}}$
1
分母为 $\sqrt{5}-\sqrt{2}$，其有理化因子为 $\sqrt{5}+\sqrt{2}$（共轭根式）。
2
分子分母同乘 $(\sqrt{5}+\sqrt{2})$： $$\frac{3}{\sqrt{5}-\sqrt{2}} \times \frac{\sqrt{5}+\sqrt{2}}{\sqrt{5}+\sqrt{2}} = \frac{3(\sqrt{5}+\sqrt{2})}{(\sqrt{5})^2-(\sqrt{2})^2}$$
3
分母：$5 - 2 = 3$，整理得：$\dfrac{3(\sqrt{5}+\sqrt{2})}{3} = \sqrt{5}+\sqrt{2}$

$\dfrac{3}{\sqrt{5}-\sqrt{2}} = \sqrt{5}+\sqrt{2}$

例题 4.3 化简根式（嵌套根号）
化简 $\sqrt{3 - 2\sqrt{2}}$
1
观察被开方数：$3 - 2\sqrt{2}$，尝试凑成完全平方形式 $(a-b)^2 = a^2 - 2ab + b^2$。
2
$3 - 2\sqrt{2} = 2 - 2\sqrt{2} + 1 = (\sqrt{2})^2 - 2 \cdot \sqrt{2} \cdot 1 + 1^2 = (\sqrt{2}-1)^2$
3
$\sqrt{3-2\sqrt{2}} = \sqrt{(\sqrt{2}-1)^2} = |\sqrt{2}-1| = \sqrt{2}-1$（因为 $\sqrt{2} > 1$）

$\sqrt{3-2\sqrt{2}} = \sqrt{2}-1$

一元二次方程

高中核心衔接

三种解法对比

以 $x^2 - 5x + 6 = 0$ 为例，展示三种解法的思路差异：

① 因式分解法

$x^2-5x+6=(x-2)(x-3)=0$

$\Rightarrow x=2$ 或 $x=3$

✅ 最快，适合能直接分解的情况

② 配方法

$x^2-5x+\frac{25}{4}=6-\frac{1}{4}$

$(x-\frac{5}{2})^2=\frac{1}{4}$

$x=\frac{5}{2}\pm\frac{1}{2}$

③ 求根公式

$a=1,b=-5,c=6$

$\Delta=25-24=1$

$x=\dfrac{5\pm1}{2}$

求根公式与判别式

求根公式（万能方法）
$$x = \frac{-b \pm \sqrt{b^2-4ac}}{2a} \quad (a \neq 0)$$
判别式 Δ 决定根的个数
$$\Delta = b^2 - 4ac$$
判别式根的情况图像与 $x$ 轴关系举例
$\Delta > 0$两个不等实根两个交点$x^2-5x+6=0$
$\Delta = 0$两个相等实根（重根）一个切点（相切）$x^2-2x+1=0$
$\Delta < 0$无实数根不相交$x^2+x+1=0$

韦达定理（根与系数关系）

设 $ax^2+bx+c=0$（$a\neq0$）的两根为 $x_1, x_2$，则：

$$x_1 + x_2 = -\frac{b}{a} \qquad x_1 \cdot x_2 = \frac{c}{a}$$

📌 韦达定理让你在不解出方程的情况下，直接得到两根之和与积，在高中数列、解析几何中大量使用。

例题 5.1 配方法完整演示
用配方法解方程 $2x^2 - 4x - 1 = 0$
1
两边除以 $2$（使 $x^2$ 系数为 $1$）：$x^2 - 2x - \dfrac{1}{2} = 0$
2
移项：$x^2 - 2x = \dfrac{1}{2}$
3
两边加上 $\left(\dfrac{-2}{2}\right)^2 = 1$（配成完全平方）：$x^2 - 2x + 1 = \dfrac{1}{2} + 1 = \dfrac{3}{2}$
4
左边变成完全平方：$(x-1)^2 = \dfrac{3}{2}$
5
两边开方：$x - 1 = \pm\sqrt{\dfrac{3}{2}} = \pm\dfrac{\sqrt{6}}{2}$，得 $x = 1 \pm \dfrac{\sqrt{6}}{2}$

$x_1 = 1 + \dfrac{\sqrt{6}}{2}$，$x_2 = 1 - \dfrac{\sqrt{6}}{2}$

例题 5.2 韦达定理应用
已知方程 $x^2 - 3x + k = 0$ 的两根之差为 $1$，求 $k$ 的值。
1
设两根为 $x_1, x_2$。由韦达定理：$x_1+x_2=3$，$x_1 x_2 = k$
2
已知 $|x_1-x_2|=1$，即 $(x_1-x_2)^2=1$
3
展开：$(x_1-x_2)^2 = (x_1+x_2)^2 - 4x_1 x_2 = 9 - 4k = 1$
4
解得 $4k = 8$，$k = 2$
5
验证：方程 $x^2-3x+2=0$，即 $(x-1)(x-2)=0$，两根 $1$ 和 $2$，差确实为 $1$ ✓

$k = 2$

例题 5.3 判别式的应用
已知方程 $x^2 + mx + m + 3 = 0$ 有两个实数根，求 $m$ 的取值范围。
1
方程有实数根的条件：判别式 $\Delta \geq 0$。这里 $a=1, b=m, c=m+3$。
2
$\Delta = m^2 - 4(m+3) \geq 0$，即 $m^2 - 4m - 12 \geq 0$
3
分解：$(m-6)(m+2) \geq 0$，解得 $m \leq -2$ 或 $m \geq 6$

$m$ 的取值范围为 $m \leq -2$ 或 $m \geq 6$，即 $(-\infty,-2] \cup [6,+\infty)$

📌 这类含参数的判别式问题在高中极为常见，是高考必考知识点。

函数思想（最核心）

⭐ 高中数学之魂

函数的三要素

① 定义域

自变量 $x$ 的所有合法取值集合。求定义域就是找「$x$ 能取哪些值」。

② 值域

函数所有可能输出的 $y$ 值的集合。往往比定义域更难求。

③ 对应法则

从 $x$ 到 $y$ 的计算规则。同一 $x$ 只能对应唯一的 $y$（竖线测试）。

求定义域的常见限制
$$\text{分母} \neq 0; \quad \text{偶次根号内} \geq 0; \quad \log \text{ 内} > 0; \quad \text{指数函数无限制}$$

函数的单调性、奇偶性

单调递增
$$x_1 < x_2 \Rightarrow f(x_1) < f(x_2)$$
单调递减
$$x_1 < x_2 \Rightarrow f(x_1) > f(x_2)$$
奇函数（原点对称）
$$f(-x) = -f(x), \quad \text{且定义域关于原点对称}$$
偶函数（$y$ 轴对称）
$$f(-x) = f(x), \quad \text{且定义域关于原点对称}$$

常见函数图像对比

■ 一次函数（单调）   ■ 二次函数（有顶点、轴对称）   ■ 反比例（奇函数、双曲线）   ■ 绝对值（偶函数、V形）

例题 6.1 求函数定义域
求函数 $f(x) = \dfrac{\sqrt{x-1}}{x-3}$ 的定义域。
1
根号内要 ≥ 0：$x-1 \geq 0$，得 $x \geq 1$
2
分母不为零：$x-3 \neq 0$，得 $x \neq 3$
3
取交集：$x \geq 1$ 且 $x \neq 3$

定义域为 $[1, 3) \cup (3, +\infty)$

例题 6.2 判断奇偶性
判断 $f(x) = x^3 - x$ 的奇偶性，并说明理由。
1
第一步：判断定义域是否关于原点对称。$f(x)$ 的定义域为 $\mathbb{R}$（全体实数），关于原点对称 ✓
2
第二步：计算 $f(-x)$：$f(-x) = (-x)^3 - (-x) = -x^3 + x = -(x^3 - x) = -f(x)$
3
由于 $f(-x) = -f(x)$ 对所有 $x \in \mathbb{R}$ 成立，且定义域关于原点对称，故 $f(x)$ 为奇函数。

$f(x) = x^3 - x$ 是奇函数，图像关于原点对称。

例题 6.3 求二次函数的值域（利用配方）
求 $f(x) = x^2 - 4x + 3$，$x \in [0, 5]$ 的值域。
1
配方：$f(x) = (x-2)^2 - 1$，顶点为 $(2, -1)$，开口向上。
2
顶点 $x=2 \in [0,5]$，故函数在 $[0,2]$ 上单调递减，在 $[2,5]$ 上单调递增。
3
计算端点值和顶点值：
  • $f(0) = 0-0+3 = 3$
  • $f(2) = 4-8+3 = -1$（最小值）
  • $f(5) = 25-20+3 = 8$（最大值）

值域为 $[-1, 8]$，最小值 $-1$（$x=2$ 时取得），最大值 $8$（$x=5$ 时取得）。

📌 求闭区间上二次函数的值域，关键是：先配方找顶点，再判断顶点是否在区间内，最后比较端点值。

两点间距离
$$|P_1P_2|=\sqrt{(x_2-x_1)^2+(y_2-y_1)^2}$$
线段中点坐标
$$M=\left(\frac{x_1+x_2}{2},\ \frac{y_1+y_2}{2}\right)$$
直线斜率（倾斜角 $\alpha$）
$$k=\frac{y_2-y_1}{x_2-x_1}=\tan\alpha \quad (x_1 \neq x_2)$$
平行与垂直
$$l_1 \parallel l_2 \Leftrightarrow k_1=k_2 \quad l_1 \perp l_2 \Leftrightarrow k_1 k_2=-1$$
例题 7.1 距离与中点综合
已知 $A(1, 2)$，$B(5, -2)$。求 $|AB|$、$AB$ 的中点 $M$，以及直线 $AB$ 的斜率。
1
距离：$|AB| = \sqrt{(5-1)^2+(-2-2)^2} = \sqrt{16+16} = \sqrt{32} = 4\sqrt{2}$
2
中点：$M = \left(\dfrac{1+5}{2}, \dfrac{2+(-2)}{2}\right) = (3,\ 0)$
3
斜率：$k = \dfrac{-2-2}{5-1} = \dfrac{-4}{4} = -1$

$|AB|=4\sqrt{2}$，中点 $M(3,0)$，斜率 $k=-1$。

例题 7.2 直线方程的建立
已知直线过点 $A(2, 1)$ 且与直线 $y = 2x - 3$ 垂直，求该直线方程。
1
已知直线的斜率 $k_1 = 2$，垂直条件：$k_1 \cdot k_2 = -1$，故 $k_2 = -\dfrac{1}{2}$
2
过点 $A(2,1)$，斜率 $-\dfrac{1}{2}$，用点斜式：$y - 1 = -\dfrac{1}{2}(x-2)$
3
整理：$y = -\dfrac{1}{2}x + 1 + 1 = -\dfrac{1}{2}x + 2$

所求直线方程为 $y = -\dfrac{1}{2}x + 2$（即 $x + 2y - 4 = 0$）

判断直角三角形

三边 $a, b, c$（最长边为 $c$），若 $a^2+b^2=c^2$ 则为直角三角形；若 $a^2+b^2 > c^2$ 则为锐角三角形；若 $a^2+b^2 < c^2$ 则为钝角三角形。

例题 8.1 勾股定理应用
等腰三角形底边长为 $10$，腰长为 $13$，求三角形的高和面积。
1
等腰三角形的高将底边平分，每半底边为 $5$，腰为 $13$，高为所求 $h$。
2
由勾股定理（认识到这是 $5:12:13$ 的勾股数组）：$h^2 + 5^2 = 13^2$，即 $h^2 = 169 - 25 = 144$，$h = 12$
3
面积 $= \dfrac{1}{2} \times 底 \times 高 = \dfrac{1}{2} \times 10 \times 12 = 60$

高为 $12$，面积为 $60$。

例题 8.2 相似三角形求边长
$\triangle ABC \sim \triangle DEF$，相似比为 $3:2$。若 $AB=9, BC=12$，求 $DE, EF$。
1
相似三角形对应边之比等于相似比，即 $\dfrac{AB}{DE} = \dfrac{BC}{EF} = \dfrac{3}{2}$
2
$DE = AB \times \dfrac{2}{3} = 9 \times \dfrac{2}{3} = 6$
3
$EF = BC \times \dfrac{2}{3} = 12 \times \dfrac{2}{3} = 8$

$DE = 6$，$EF = 8$。

📌 面积之比 = 相似比的平方 = $9:4$。这是高中中很常用的结论。

单位圆

三角函数的定义（直角三角形）

$$\sin\theta = \frac{\text{对边}}{斜边}, \quad \cos\theta = \frac{\text{邻边}}{斜边}, \quad \tan\theta = \frac{\text{对边}}{\text{邻边}} = \frac{\sin\theta}{\cos\theta}$$

🧠 记忆口诀："正弦对斜，余弦邻斜，正切对邻"。英文：SOH-CAH-TOA（Sine=Opposite/Hypotenuse 等）

特殊角三角函数值（必须背熟）

角度弧度$\sin$$\cos$$\tan$
$0°$$0$$0$$1$$0$
$30°$$\dfrac{\pi}{6}$$\dfrac{1}{2}$$\dfrac{\sqrt{3}}{2}$$\dfrac{\sqrt{3}}{3}$
$45°$$\dfrac{\pi}{4}$$\dfrac{\sqrt{2}}{2}$$\dfrac{\sqrt{2}}{2}$$1$
$60°$$\dfrac{\pi}{3}$$\dfrac{\sqrt{3}}{2}$$\dfrac{1}{2}$$\sqrt{3}$
$90°$$\dfrac{\pi}{2}$$1$$0$不存在
$180°$$\pi$$0$$-1$$0$

📌 记忆技巧：$\sin$ 值 $0° \to 90°$ 依次为 $0, \dfrac{1}{2}, \dfrac{\sqrt{2}}{2}, \dfrac{\sqrt{3}}{2}, 1$，即 $\dfrac{\sqrt{0}}{2}, \dfrac{\sqrt{1}}{2}, \dfrac{\sqrt{2}}{2}, \dfrac{\sqrt{3}}{2}, \dfrac{\sqrt{4}}{2}$。$\cos$ 反过来。

单位圆：三角函数的几何定义

基本三角恒等式

勾股恒等式（最重要）
$$\sin^2\theta + \cos^2\theta = 1$$
由此推出
$$\sin^2\theta = 1 - \cos^2\theta$$ $$1 + \tan^2\theta = \frac{1}{\cos^2\theta} \quad (\cos\theta \neq 0)$$

弧度与角度互换

$$\text{弧度} = \text{角度} \times \frac{\pi}{180°}, \quad \text{角度} = \text{弧度} \times \frac{180°}{\pi}$$

✅ 高中三角函数全部用弧度制。理解弧度：$\pi$ 弧度 $= 180°$，$2\pi$ 弧度 $= 360°$（一圈）。

例题 9.1 利用恒等式化简
已知 $\sin\theta = \dfrac{3}{5}$，且 $\theta$ 是第二象限角，求 $\cos\theta$ 和 $\tan\theta$。
1
由 $\sin^2\theta + \cos^2\theta = 1$：$\cos^2\theta = 1 - \left(\dfrac{3}{5}\right)^2 = 1 - \dfrac{9}{25} = \dfrac{16}{25}$
2
$\theta$ 在第二象限，$\cos\theta < 0$，故 $\cos\theta = -\dfrac{4}{5}$
3
$\tan\theta = \dfrac{\sin\theta}{\cos\theta} = \dfrac{3/5}{-4/5} = -\dfrac{3}{4}$

$\cos\theta = -\dfrac{4}{5}$，$\tan\theta = -\dfrac{3}{4}$

📌 各象限符号规律：第一象限三者均正；第二 $\sin$ 正；第三 $\tan$ 正；第四 $\cos$ 正。口诀：「一全正，二正弦，三正切，四余弦」

例题 9.2 特殊角计算
计算：$\sin 30° \cdot \cos 60° + \cos 30° \cdot \sin 60°$
1
代入特殊值：$\sin 30° = \dfrac{1}{2}$，$\cos 60° = \dfrac{1}{2}$，$\cos 30° = \dfrac{\sqrt{3}}{2}$，$\sin 60° = \dfrac{\sqrt{3}}{2}$
2
$= \dfrac{1}{2} \times \dfrac{1}{2} + \dfrac{\sqrt{3}}{2} \times \dfrac{\sqrt{3}}{2} = \dfrac{1}{4} + \dfrac{3}{4} = 1$

结果为 $1$

📌 该式其实正是高中的正弦和角公式：$\sin(A+B)=\sin A\cos B+\cos A\sin B$，即 $\sin(30°+60°)=\sin 90°=1$。你已经在预习高中内容了！

集合初步

高中第一章

集合的基本概念与符号

属于
$3 \in \{1,2,3\}$（$3$ 是集合的元素）
不属于
$4 \notin \{1,2,3\}$
子集
$A \subseteq B$（$A$ 的每个元素都在 $B$ 中）
真子集
$A \subsetneq B$（$A \subseteq B$ 且 $A \neq B$）
空集
$\emptyset$（不含任何元素，是任意集合的子集）
全集
$U$（研究范围内的所有元素之集合）
集合的两种表示法
$$A = \{1, 2, 3, 4, 5\} \quad \text{（列举法）}$$ $$B = \{x \mid x^2 \leq 4,\ x \in \mathbb{Z}\} = \{-2,-1,0,1,2\} \quad \text{（描述法）}$$

集合的三大运算

交集 $A \cap B$

同时属于 $A$ 和 $B$ 的元素组成的集合（公共部分）。

$\{1,2,3\}\cap\{2,3,4\}=\{2,3\}$

并集 $A \cup B$

属于 $A$ 或属于 $B$ 的所有元素（不重复计数）。

$\{1,2,3\}\cup\{2,3,4\}=\{1,2,3,4\}$

补集 $\complement_U A$

在全集 $U$ 中，不属于 $A$ 的所有元素。

若 $U=\{1,2,3,4,5\}$，$A=\{1,2\}$，则 $\complement_U A=\{3,4,5\}$

韦恩图可视化

例题 10.1 集合运算（含不等式）
已知全集 $U = \mathbb{R}$，$A = \{x \mid -1 \leq x < 3\}$，$B = \{x \mid x < 1\}$，求 $A \cap B$，$A \cup B$，$\complement_U A$。
1
在数轴上标出：$A = [-1, 3)$，$B = (-\infty, 1)$
2
交集（公共区间）：两段区间的重叠部分为 $[-1, 1)$ $$A \cap B = \{x \mid -1 \leq x < 1\} = [-1, 1)$$
3
并集（合并覆盖）：将两区间合并（不重叠部分 $B$ 还包含 $(-\infty,-1)$） $$A \cup B = \{x \mid x < 3\} = (-\infty, 3)$$
4
补集（$A$ 之外的实数）：$A=[-1,3)$，其补集为两侧 $$\complement_U A = \{x \mid x < -1 \text{ 或 } x \geq 3\} = (-\infty, -1) \cup [3, +\infty)$$

$A \cap B = [-1, 1)$，$A \cup B = (-\infty, 3)$，$\complement_U A = (-\infty,-1)\cup[3,+\infty)$

例题 10.2 含参数的集合（高考常考）
已知 $A = \{1, 2, a\}$，$B = \{2, a^2\}$，且 $B \subseteq A$，求实数 $a$ 的值。
1
$B \subseteq A$ 意味着 $B$ 中每个元素都必须在 $A$ 中，已知 $2 \in A$ ✓，所以只需 $a^2 \in A$。
2
$A$ 中的元素为 $1, 2, a$，所以 $a^2$ 必须等于 $1$、$2$ 或 $a$ 之一。
3
情况一：$a^2 = 1$，则 $a = 1$ 或 $a = -1$。 若 $a=1$：$A=\{1,2,1\}$ 有重复元素，集合中元素互异，矛盾，舍去。 若 $a=-1$：$A=\{1,2,-1\}$，$B=\{2,1\}$，检查 $B \subseteq A$：$2\in A$ ✓，$1\in A$ ✓，成立 ✓
4
情况二：$a^2 = 2$，则 $a = \pm\sqrt{2}$。 若 $a=\sqrt{2}$：$A=\{1,2,\sqrt{2}\}$，$B=\{2,2\}$ 有重复，矛盾，舍去。 若 $a=-\sqrt{2}$：$A=\{1,2,-\sqrt{2}\}$，$B=\{2,2\}$ 有重复，矛盾，舍去。
5
情况三：$a^2 = a$，则 $a(a-1)=0$，$a=0$ 或 $a=1$。 若 $a=0$：$A=\{1,2,0\}$，$B=\{2,0\}$，$B\subseteq A$ ✓ 若 $a=1$：重复，已舍。

$a = -1$ 或 $a = 0$

📌 含参数集合题的关键：① 考虑所有情况；② 每种情况都要验证「元素互异」；③ 检查题目条件是否满足。
